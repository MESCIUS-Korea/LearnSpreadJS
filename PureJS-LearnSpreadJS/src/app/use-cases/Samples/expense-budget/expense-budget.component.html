비용경비 예산

비용 예산은 기업의 판매량과 이익에 영향을 미치는 중요한 요소입니다.

Spread.Sheets를 사용하여 관련 데이터를 표에 표시하고 조건부 형식을 사용하여 데이터를 시각화 할 수 있습니다.

예를 들어, IconSetRule을 사용하여 다른 상태를 표시하고, DataBarRule을 사용하여 모든 예산을 표시하는 등의 작업을 수행 할 수 있습니다

var sd = JSON.parse(data);
spread.suspendPaint();
spread.fromJSON(sd[0]);
spread.resumePaint();

결과화면과 함께 HTML, JS 코드도 확인이 가능합니다.
