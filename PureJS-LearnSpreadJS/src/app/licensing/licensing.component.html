Spread.Sheets 배포 라이센싱

※ 서버 배포를 위해, 도메인 또는 IP주소 당 추가적인 라이선스 비용이 발생합니다. (별도 문의)

정식 버전의 Spread.Sheets 및 Spread.Views를 서버에 배포하기 위한 라이선스 키입니다. 서버에 SpreadJS 컨트롤을 배포하기 위해서는 별도의 라이선스 키를 추가로 발급받으셔야 합니다.

GrapeCity Korea(sales-kor@grapecity.com)로 아래의 정보를 주시면 해당 정보의 배포용 라이선스 키를 발급해 드립니다.


Product Key (구매 시 메일로 받으신, 고유의 키)

회사명

연락처

도메인/IP주소


다음 단계에 따라 라이센스 파일을 생성할 수 있습니다 :

  1. 다운로드 한 SpreadSheetsLicensedX.XX.XXXX.X.zip을 평가판과 동일한 위치에서 압축을 풀어 해당 위치에있는 기존 파일을 모두 덮어씁니다.

  2. 배포용 Spread.Sheets 라이선스에 대한 참조를 추가하십시오. WorkBook을 초기화하기 전에 참조를 추가하십시오.

    <script src="./scripts/evaluation/gc.spread.sheets.all.10.0.0.min.js" type="application/javascript"></script>
<script>
   GC.Spread.Sheets.LicenseKey = "xxx";
</script>

    또는,

    <script src="./scripts/evaluation/gc.spread.sheets.all.10.0.0.min.js" type="application/javascript"></script>
<script src="./scripts/evaluation/licensekey.js" type="application/javascript"></script>
<script>     
   window.onload = function () {
      var activeSheet = spread.getActiveSheet();
   }
</script>

    licensekey.js file:

    GC.Spread.Sheets.LicenseKey = "xxxxxx";

Note: 평가판 또는 프로덕션 라이선스가 없는 경우 Spread.Sheets은 로컬 컴퓨터에서만 사용할 수 있습니다.


Spread.Sheets 디자이너

Spread.Sheets 디자이너는 라이센스 키없이 30 일간 평가판으로 사용할 수 있습니다.

잠금 아이콘을 선택하고 재품 구매시 받은 디자이너 용, 라이센스 키를 입력하여 Spread.Sheets Designer의 잠금을 해제하십시오.


Spread.Sheets Excel 가져오기/내보내기 컴포넌트

새로운 Importer () 또는 새로운 Exporter () 인스턴스 앞에 라이센스 키를 GrapeCity.Spreadsheet.License.License Key로 설정하십시오:

GrapeCity.Spread.Sheets.License.LicenseKey = "your key";

웹 응용 프로그램을 만들고 스크립트 파일을 사용하는 경우 Spread.Sheets 스크립트 파일을 참조한 후에 라이센스 키를 추가해야합니다:

<script>GC.Spread.Sheets.LicenseKey = "your key";</script>

클라이언트 사이드 Excel 가져오기/내보내기 옵션은 다운받으신 Spread.Sheets에 포함되어 있습니다.

혹시, 서버 사이드의 Windows IIS 서버 용 Excel 가져오기/내보내기 구성요소가 필요하신 분은 http://sphelp.grapecity.com/devchannel/에서 다운받을 수 있습니다.
