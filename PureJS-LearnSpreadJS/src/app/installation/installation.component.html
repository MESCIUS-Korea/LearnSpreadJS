SpreadJS - Spread.Sheets 설치안내

Spread.Sheets를 설치하려면 다음 단계를 따르면 됩니다.

Spread.Sheets을 설치하기 위해,

1.다운로드 한 ZIP 파일 (Spread.Sheetsxxxx.zip)을 시스템의 원하시는 디렉토리에 저장한 다음 파일을 해당 디렉토리에 압축을 풉니다.


기본 Spread.Sheets 폴더 (Windows 기준)

Install Directory File Description
\Spread.Sheets.xxxx   Main directory for product
\Spread.Sheets.xxxx\css   CSS files
\Spread.Sheets.xxxx\samples   Sample pages for the widget
\Spread.Sheets.xxxx\scripts gc.spread.sheets.all.xxxx.min Minimized js file for widget which includes all functions
\Spread.Sheets.xxxx\definition GC.spread.sheets.d.ts TypeScript API definition file

  gc.spread.excelio.xxxxx.min Client-side Excel import and export support files
  gc.spread.sheets.migration.xxxxx.min Migration file for backward compatibility
\Spread.Sheets.xxxx\scripts\resources ja\gc.spread.sheets.resources.ja.xxxxx.min
zh\gc.spread.sheets.resources.zh.xxxx.min 		Japanese and Chinese resource files

Spread.Sheets 디자이너를 설치하기 위해,

1.다운로드 한 ZIP 파일 (SpreadSheetsDesigner.zip)을 시스템의 원하시는 디렉토리에 저장한 다음 파일을 해당 디렉토리에 압축을 풉니다.

2.사용자 환경에 맞는 설치 파일을 실행하십시오.

3.디자이너를 실행하고 자물쇠 아이콘을 선택한 다음 라이센스 키를 입력하여 Spread.Sheets 디자이너의 잠금을 해제하십시오.

Note: 체험판의 디자이너를 설치한 분들은, 잠금 해제를 위해서는 정식 버전의 디자이너로 재설치 해주시길 바랍니다.



Platform Setup
Windows Spread.Sheets-Designer-Evaluation-Setup.10.0.0.exe or Spread.Sheets-Designer-Production-Setup.10.0.0.exe
Mac Spread.Sheets-Designer-Evaluation-10.0.0.dmg or Spread.Sheets-Designer-Production-10.0.0.dmg
Linux Spread.Sheets-Designer-Evaluation-10.0.0-x86_64.AppImage or Spread.Sheets-Designer-Production-10.0.0-x86_64.AppImage
