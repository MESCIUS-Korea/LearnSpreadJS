플로팅 객체 동작 | Floating Object Actions

1. 플로팅 객체는 드래그 드롭, 크기 조정, 이동, 복사, 붙여 넣기 등의 다양한 동작을 지원합니다.

2. 다음 키는 플로팅 객체로 작업하기 위한 작업을 제공합니다.

  • Esc : 여러 개의 플로팅 객체를 선택한 다음 Esc 키를 누르면 모든 플로팅 객체가 선택 취소됩니다.
  • Delete: 여러 개의 플로팅 객체를 선택한 다음 Delete 키를 누르면 선택한 플로팅 객체가 삭제됩니다.
  • Tab : 플로팅 객체를 선택한 다음 Tab 키를 눌러 다음 플로팅 객체를 탐색하고 선택합니다.
  • Shift + Tab :플로팅 객체를 선택한 다음 Shift + Tab을 눌러 이전 플로팅 객체를 탐색하고 선택하십시오.
  • Ctrl + X : 여러 개의 플로팅 객체를 선택한 다음 Ctrl + X를 누르면 선택한 플로팅 객체가 클립 보드로 지워집니다.
  • Ctrl + C : 여러 개의 플로팅 객체를 선택한 다음 Ctrl + C를 누르면 선택한 플로팅 객체가 클립 보드로 복사됩니다.
  • Ctrl + V : 셀을 선택한 다음 Ctrl + V를 눌러 클립 보드에서 붙여 넣습니다.
  • Ctrl + A : 플로팅 객체를 선택한 다음 Ctrl + A를 눌러 모든 플로팅 객체를 선택하십시오.
  • Up : 플로팅 객체를 선택한 다음 위쪽 화살표를 누르면 선택한 플로팅 객체가 위로 이동합니다.
  • Down : 플로팅 객체를 선택한 다음 아래쪽 화살표를 눌러 선택한 객체를 아래로 이동합니다.
  • Left : 플로팅 객체를 선택한 다음 왼쪽 화살표를 눌러 선택한 객체를 왼쪽으로 이동합니다.
  • Right : 일부 플로팅 객체를 선택한 다음 오른쪽 화살표를 눌러 선택한 플로팅 객체를 오른쪽으로 이동하십시오.

3. 마우스 동작을 사용하여 플로팅 객체와 상호 작용할 수도 있습니다. 플로팅 객체를 선택하면 위치 표시기와 8 개의 크기 조정 표시기가 표시됩니다. 크기 조정 표시기를 드래그하여 플로팅 객체의 크기를 조정할 수 있습니다. 또한 플로팅 객체를 드래그하여 다른 위치에 놓을 수 있습니다.

4. Ctrl 키를 누른 상태에서 마우스로 플로팅 객체를 드래그하여 복사하여 붙여 넣을 수 있습니다. 마우스를 놓으면 드래그 된 플로팅 객체가 복사되고 붙여 넣어집니다.
