상자그림 스파크라인 | Boxplot Sparkline

BoxPlotSparkline 함수를 사용하여 boxplot 스파크 라인을 만들 수 있습니다.

= BOXPLOTSPARKLINE (points, boxPlotClass ?, showAverage?, scaleStart?, scaleEnd ?, acceptableStart ?, acceptableEnd, colorScheme?, style ?, vertical?).

이 함수에는 다음과 같은 매개 변수가 있습니다.

points : "A1 : A4"와 같이 설명 할 모든 값을 포함하는 셀 범위를 나타내는 레퍼런스입니다.

boxPlotClass : (선택 사항) boxPlot 클래스입니다. 다음 중 하나입니다.

  • 5ns (기본값)
  • 7ns
  • tukey
  • bowley
  • sigma 3

showAverage : (선택 사항) 평균을 표시할지 여부를 나타내는 Boolean 값입니다. 기본값은 false입니다.

scaleStart : (선택 사항) 스파크 라인의 최소 경계를 나타내는 숫자 또는 레퍼런스 (예 : 1 또는 "A6"). 기본값은 모든 값 중 최소값입니다.

scaleEnd : (선택 사항) 스파크 라인의 최대 경계를 나타내는 숫자 또는 레퍼런스 (예 : 8 또는 "A7"). 기본값은 모든 값의 최대 값입니다.

acceptableStart : (선택 사항) 허용되는 행의 시작을 나타내는 숫자 또는 레퍼런스 (예 : 3 또는 "A8"). 기본값은 null입니다.

acceptableEnd : (선택 사항) 5 또는 "A9"와 같이 허용되는 줄의 끝을 나타내는 숫자 또는 레퍼런스입니다. 기본값은 null입니다.

colorScheme : (선택 사항) 스파크 라인 상자의 색상을 나타내는 문자열. 기본값은 "# D2D2D2"입니다.

style : (선택 사항) boxPlot 스파크 라인의 스타일을 나타내는 숫자 또는 레퍼런스입니다. 다음 중 하나입니다.

  • 0 : (기본값) 클래식 스타일 - 수염 모양이 선이고 바깥 값이 원입니다.
  • 1 : Neo 스타일 - 수염 모양이 직사각형이며 이상 치가 선입니다.

vertical : (선택 사항) 스파크 라인을 세로로 표시할지 여부를 나타내는 Boolean 값입니다. 기본값은 false입니다.
