스파크라인 설정 | Sparkline Setting

1. 스파크 라인을 만든 후에는 표시되는 값의 포인트 (예 : 높음, 낮음, 우선, 마지막 또는 음수 값), 스파크 라인 유형 변경 (선, 열 또는 WinLoss), 스타일 적용, 가로축을 표시할지 여부를 제어합니다.

2. 마커 중 일부 또는 전부를 표시하여 라인 스파크 라인의 개별 데이터 마커 (값)를 강조 표시 할 수 있습니다.

  • showFirst :이 스파크 라인 그룹의 각 스파크 라인에 대해 첫 번째 데이터 포인트의 형식이 다르게 지정되는지 여부
  • showHigh :이 스파크 라인 그룹의 각 스파크 라인에 대해 가장 높은 값을 가진 데이터 포인트가 다르게 포맷되었는지 여부
  • showLast :이 스파크 라인 그룹의 각 스파크 라인에 대해 마지막 데이터 포인트가 다르게 포맷되었는지 여부
  • showLow : 가장 낮은 값을 가진 데이터 포인트가 이 스파크 라인 그룹의 각 스파크 라인에 대해 다르게 포맷되는지 여부
  • showNegative : 음의 데이터 포인트가 이 스파크 라인 그룹의 각 스파크 라인에 대해 다르게 포맷되는지 여부
  • showMarkers :이 스파크 라인 그룹의 각 스파크 라인에 데이터 마커가 표시되는지 여부

3. 다음 방법을 사용하여 스파크 라인의 스타일과 형식을 변경할 수 있습니다.

  • axisColor : 축의 색
  • firstMarkerColor :이 스파크 라인 그룹의 각 스파크 라인에 대한 첫 번째 데이터 포인트의 색상
  • highMarkerColor :이 스파크 라인 그룹의 각 스파크 라인에 대해 가장 높은 데이터 포인트의 색상
  • lastMarkerColor :이 스파크 라인 그룹의 각 스파크 라인에 대한 마지막 데이터 포인트의 색상
  • lowMarkerColor :이 스파크 라인 그룹의 각 스파크 라인에 대한 가장 낮은 데이터 포인트의 색상
  • markersColor :이 스파크 라인 그룹의 각 스파크 라인에 대한 데이터 마커의 색상
  • negativeColor :이 스파크 라인 그룹의 각 스파크 라인에 대한 음수 데이터 포인트의 색상
  • seriesColor :이 스파크 라인 그룹의 각 스파크 라인 색상

4. 스파크 라인은 추가 설정을 제공합니다. 예를 들어 차트의 데이터 계열에 빈 값이있는 경우가 있습니다. 다음 예와 같이 displayEmptyCellsAs 옵션을 사용하여 빈 셀을 표시하는 방법을 제어 할 수 있습니다. 

var setting = new GC.Spread.Sheets.Sparklines.SparklineSetting();
setting.options.displayEmptyCellsAs = GC.Spread.Sheets.Sparklines.EmptyValueStyle.gaps;
setting.options.rightToLeft = true;
setting.options.displayHidden = false;
setting.options.displayXAxis = false
setting.options.lineWeight = 2;
setting.options.manualMax = 3;
setting.options.manualMin = 1;
setting.options.markersColor = 'Magenta';
setting.options.maxAxisType = GC.Spread.Sheets.Sparklines.SparklineAxisMinMax.custom;
setting.options.minAxisType = GC.Spread.Sheets.Sparklines.SparklineAxisMinMax.individual;

5. 다음 예제는 이러한 설정을 적용하는 방법을 보여줍니다. 

var setting = new GC.Spread.Sheets.Sparklines.SparklineSetting();
setting.options.showFirst = true;
setting.options.showHigh = true;
setting.options.displayXAxis = true;
setting.options.axisColor = 'Cyan';
var sparkline = sheet.getSparkline(11, 0);
sparkline.setting(setting);
