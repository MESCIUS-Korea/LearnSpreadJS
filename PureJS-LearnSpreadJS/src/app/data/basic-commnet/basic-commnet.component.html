사용자 코멘트 | Custom Comment

1. 스프레드 시트에서 주석을 사용하여 시트에 노트를 추가 할 수 있습니다. 주석을 사용하면 포함 된 데이터의 추가 컨텍스트를 제공하여 시트를 더 쉽게 이해할 수 있습니다.

2. 시트에 주석을 추가하려면 먼저 주석을 작성하고 주석을 설정하십시오. 그런 다음 add 메소드를 사용하여 지정된 셀에 대한 주석을 설정할 수 있습니다. 예 : 

sheet.comments.add(5, 5, 'new comment!');

다음 예제와 같이 추가 한 주석을 제거하려는 경우가 있습니다. 

sheet.comments.remove(5, 5);

3. 셀에 주석을 추가 한 후 get 메서드를 사용하여 주석을 가져 오거나 all 메서드를 사용하여 시트의 모든 주석을 가져올 수 있습니다. 

var comment = sheet.comments.get(5, 5);
var comments = sheet.comments.all();

샘플코드:

결과화면과 함께 HTML, JS 코드도 확인이 가능합니다.
