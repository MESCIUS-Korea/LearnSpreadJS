Compatible 스파크라인 | Compatible Sparkline

함수에는 다음과 같은 매개 변수가 있습니다.

data: 스파크 라인 데이터를 나타내는 "A1 : C3"와 같은 범위 레퍼런스.

dataOrientation: 스파크 라인 데이터 방향을 나타내는 숫자입니다. 다음 중 하나입니다.

  • vertical - 0
  • horizontal - 1

dataAxisData : (선택 사항) 스파크 라인 날짜 축 데이터를 나타내는 "D1 : F3"와 같은 범위 레퍼런스.

dataAxisOrientation : (선택 사항) 스파크 라인 날짜 축 방향을 나타내는 숫자입니다. 다음 중 하나입니다.

  • vertical - 0
  • horizontal - 1

setting : (선택 사항) JSON 형식의 문자열. 각 항목은 GC.Spread.Sheets.Sparklines.SparklineSetting과 동일한 정의를 유지합니다.
