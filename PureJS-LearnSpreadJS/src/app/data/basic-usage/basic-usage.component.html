기초 사용법 | Basic Usage

1. 태그를 사용하여 개인 데이터를 저장할 수 있습니다. Spread.Sheets는 셀, 행, 열 및 시트에 대한 태그를 지원합니다.

Sheet.tag (value?) : 시트의 태그를 가져 오거나 설정합니다.

Sheet.getTag (row, col, sheetArea?) / Sheet.setTag (row, col, tag, sheetArea?) : 태그를 가져 오거나 설정합니다.

  • sheet : 행이 -1이고 col이 -1 일 때
  • row : col이 -1이고 row> = 0 인 경우
  • column : row가 -1이고 col> = 0 일 때
  • cell : 행> = 0 및 col> = 0 일 때

CellRange.tag(value?) : 셀의 태그를 가져 오거나 설정합니다.

2. 태그가 설정되면 다음 이벤트가 발생합니다.

  • CellRange.tag (value)때 CellChanged 이벤트
  • CellRange.tag (value)때 RowChanged 이벤트
  • CellRange.tag (value)때 ColumnChanged 이벤트

3. 태그를 null 또는 undefined로 설정하거나 Sheet.clear (row, column, rowCount, columnCount, sheetArea, StorageType.tag)를 사용하여 태그를 지울 수 있습니다. 예 :

예 

sheet.tag('Sheet tag');
sheet.setTag(3, -1, 'Row tag');
sheet.getRange(5, -1, 1, -1).tag('Row tag too');
sheet.getRange(-1, 4, -1, 3).tag('Column tag');
sheet.getCell(0, 0).tag('A1 tag');
sheet.getRange(6, 0, 4, 3).tag('Cells tag');
sheet.clear(6, 0, 9, 2, spreadNS.SheetArea.viewport,spreadNS.StorageType.tag);
