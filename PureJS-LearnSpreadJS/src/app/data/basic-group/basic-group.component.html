기본그룹 | Basic Group

1. 그룹화하고 요약하려는 데이터 목록이있는 경우 행 또는 열의 개요를 작성할 수 있습니다. 행 그룹 패널은 행 헤더의 왼쪽에 있습니다. 열 그룹 패널이 열 헤더글 위에 있습니다.

2. Spread.Sheets에 그룹을 추가하려면 다음 예제와 유사한 코드를 사용하십시오. 

var spread = $('#ss').data('workbook');
var sheet = spread.getActiveSheet();
sheet.rowOutlines.group(1, 10);
sheet.columnOutlines.group(1, 5);

3. 다른 그룹에 그룹을 추가 할 수 있습니다. 각 그룹에 대해 하나씩 여러 레벨의 아웃라인를 만들 수 있습니다. 아웃라인 기호에서 더 높은 숫자로 표시되는 각 Inner레벨은 아웃라인 기호에서 더 낮은 숫자로 표시되는 이전 외부 수준에 대한 상세 데이터를 표시합니다. 요약 행 또는 열을 빠르게 표시하거나 각 그룹의 세부 정보 데이터를 표시하려면 아웃라인을 사용하십시오.

4. 더 이상 그룹이 필요하지 않으면 ungroup 또는 ungroupRange 메소드를 사용하여 그룹을 제거하십시오. 예 : 

var spread = $('#ss').data('workbook');
var sheet = spread.getActiveSheet();
sheet.rowOutlines.ungroupRange(1, 10);
sheet.columnOutlines.ungroupRange(1, 5);

샘플코드:

결과화면과 함께 HTML, JS 코드도 확인이 가능합니다.
