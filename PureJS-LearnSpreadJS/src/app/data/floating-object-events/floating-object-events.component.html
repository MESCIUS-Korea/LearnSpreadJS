플로팅 객체 이벤트 | Floating Object Events

1. Spread.Sheets는 발생한 이벤트에 반응 할 수 있도록 시트에 플로팅 객체에 대한 이벤트를 제공합니다.

2. FloatingObject 플로팅 객체에는 FloatingObjectSelectionChanged (플로팅 객체 선택이 변경된 경우) 및 FloatingObjectChanged (플로팅 객체가 변경된 경우)의 두 가지 이벤트가 있습니다. 

sheet.bind(GC.Spread.Sheets.Events.FloatingObjectSelectionChanged, function(e, args) {
   // do something
});
sheet.bind(GC.Spread.Sheets.Events.FloatingObjectChanged, function(e, args) 	
{
   // do something
});

3. 그림 플로팅 객체에는 두 가지 종류의 이벤트가 있습니다. PictureSelectionChanged (플로팅 객체 선택이 변경된 경우) 및 PictureChanged (그림이 변경된 경우) 

sheet.bind(GC.Spread.Sheets.Events.PictureSelectionChanged, function(e, args) {
    // do something
});
sheet.bind(GC.Spread.Sheets.Events.PictureChanged, function(e, args) {
    // do something
});
