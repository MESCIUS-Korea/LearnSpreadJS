사용자 플로팅 객체 | Custom Floating Object

1. 위치 변경, 크기 조정 등 플로팅 개체를 사용자 지정할 수 있습니다.

2. 플로팅 개체의 위치와 크기를 변경할 수 있습니다. 위치와 크기는 두 가지 방법으로 설정할 수 있습니다. 예 : 

floatingObject.startRow(1);
floatingObject.startColumn(1);
floatingObject.startRowOffset(0);
floatingObject.startColumnOffset(0);
floatingObject.endRow(4);
floatingObject.endColumn(4);
floatingObject.endRowOffset(0);
floatingObject.endColumnOffset(0);
// or
var point = new GC.Spread.Sheets.Point(62, 20);
floatingObject.position(point);
floatingObject.width(186);
floatingObject.height(60);

3. 가끔 행의 높이 또는 열 너비의 크기를 조정할 것이므로 플로팅 개체의 위치나 크기가 변경되는 것을 원하지는 않습니다. 이 경우 dynamicMove 및 dynamicSize 메서드를 사용하십시오. 예 : 

floatingObject.dynamicMove(false);
floatingObject.dynamicSize(false);

dynamicMove 메서드가 false이고 dynamicSize 메서드가 true이면 어느 것도 영향을 끼치지 않습니다.

4. UI 상호 작용에 따라 플로팅 객체를 변경하지 않으려면 isLocked 메서드를 사용하여 고정시킵니다. 잠그고 싶으면 먼저 시트를 잠급니다.

5. 하나의 플로팅 오브젝트가 다른 오브젝트와 겹치면 zIndex 메소드를 사용하여 위에서 아래로 순서를 변경할 수 있습니다. 예 : 

var zIndex = floatingObject.zIndex('f1');
floatingObject.zIndex('f1', zIndex + 1);

결과화면과 함께 HTML, JS 코드도 확인이 가능합니다.
