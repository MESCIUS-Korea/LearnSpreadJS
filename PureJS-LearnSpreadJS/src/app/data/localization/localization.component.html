지역화 | Localization

1. Spread.Sheets은 영어 문화 (기본값), 중국어 (해당 리소스 파일 필요) 및 일본어 (해당 리소스 파일 필요)에 대한 기본 지원을 제공합니다. 이를 사용하여 지역화 작업을 지원할 수 있습니다.

2. 지역화를 지원하려면 스프레드 링크 아래의 문서 머리 부분에 적절한 js 파일 링크를 추가해야합니다. 예 : 

<head>
    ...
    <script src='.../spreadjs/gc.spread.sheets.all.x.xx.xxxxx.x.min.js' type='text/javascript'></script>
    <script src='.../spreadjs/resources/ja/gc.spread.sheets.resources.ja.x.xx.xxxxx.x.min.js' type='text/javascript'></script>
</head>

3. 다음 방법 중 하나를 선택하여 지역화를 지원할 수 있습니다.

Culture meta

메타 태그를 사용하여 문화권을 설정할 수 있습니다. 

<head>
    ...
    <meta name="spreadjs culture" content="ja-jp" />
    ...
</head>

코드로 변경

GC.Spread.Common.CultureManager.culture 메서드를 사용하여 문화권을 변경할 수 있습니다. 

// init spread
GC.Spread.Common.CultureManager.culture("ja-jp");
// GC.Spread.Common.CultureManager.culture("zh-cn");

샘플코드:

결과화면과 함께 HTML, JS 코드도 확인이 가능합니다.
