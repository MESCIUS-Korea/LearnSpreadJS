사용자 정의 테이블 | Custom Table

1. 일부 테이블을 추가 한 후 테이블을 사용자 정의하여 다른 범위로 이동하거나 크기를 조정하거나 제거하려는 경우가 있습니다.

2. 테이블을 얻으려면 find, findByName 또는 all 메소드를 사용하십시오. 예 : 

var spread = $('workbook');
var sheet = spread.getActiveSheet();
var  table = sheet.tables.add('table1', 0, 0, 4, 4, tableStyle);
table === sheet.tables.find(0,0);  // true
table === sheet.tables.findByName('table1');
 // get all the tables in this sheet.
var  tables = sheet.tables.all();

3. 테이블 위치를 변경하거나 테이블 크기를 변경하려면 move 메서드를 사용하여 위치를 변경하고 resize 메서드를 사용하여 크기를 변경할 수 있습니다. 

sheet.tables.move(table, 1, 1);
sheet.tables.resize(table, 5, 5);

4. 테이블이 필요하지 않으면 remove 메소드를 사용하여 테이블을 제거 할 수 있습니다. 예 : 

sheet.tables.remove(table);
