Pareto 스파크라인 | Pareto Sparkline

ParetoSparkline 함수를 사용하여 Pareto 스파크 라인을 만들 수 있습니다.

=PARETOSPARKLINE(points, pointIndex, colorRange?, target?, target2?, highlightPosition?, label?, vertical?).

이 함수에는 다음과 같은 매개 변수가 있습니다.

points : 설명 할 모든 값을 포함하는 셀 범위를 나타내는 레퍼런스 (예 : "B2 : B7")

pointIndex : 점의 세그먼트 인덱스를 나타내는 숫자 또는 레퍼런스이며 pointIndex> = 1 (예 : 1 또는 "D2")

colorRange : (선택 사항) 세그먼트 상자의 색상을 포함하는 셀 범위를 나타내는 레퍼런스 (예 : "D2 : D7") 기본값은 null입니다.

target : (선택 사항) "target"행 위치를 나타내는 숫자 또는 레퍼런스 (예 : 0.5). 기본값은 null입니다.

target2 : (선택 사항) "target2"행 위치를 나타내는 숫자 또는 레퍼런스 (예 : 0.5). 기본값은 null입니다.

highlightPosition : (선택 사항) 빨간색으로 색칠 될 세그먼트의 순위를 나타내는 숫자 또는 레퍼런스 (예 : 3). 기본값은 null입니다.

label : (선택 사항) 누적 백분율 (label = 1) 또는 단일 백분율 (label = 2) 또는 none (label = 0)으로 표시되는지 여부를 나타내는 숫자입니다. 기본값은 none입니다.

vertical : (선택 사항) 상자의 방향이 수직인지 여부를 나타내는 Boolean 값입니다. 기본값은 false입니다.
