캐스케이드 스파크라인 | Cascade Sparkline

수식에서 CascadeSparkline 함수를 사용하여 Cascade 스파크라인을 만들 수 있습니다.

=CASCADESPARKLINE(pointsRange, pointIndex, labelsRange?, minimum?, maximum?, colorPositive?, colorNegative?, vertical?).

이 함수에는 다음과 같은 매개 변수가 있습니다.

pointsRange: "B2 : B8"과 같이 설명 할 모든 값을 포함하는 셀 범위를 나타내는 레퍼런스입니다.

pointIndex: Point의 Point 인덱스를 나타내는 숫자 또는 참조 및 pointIndex> = 1 (예 : 1 또는 "D2")

labelsRange: (선택 사항) 설명 할 모든 레이블을 포함하는 셀 범위를 나타내는 레퍼런스(예 : "A2 : A8"). 기본값은 레이블이 없습니다.

minimum: (선택 사항) 표시 영역의 최소값을 나타내는 숫자 또는 레퍼런스입니다 (예 : -2000). 기본값은 합계의 최소값 (점의 합계)입니다.

maximum: (선택 사항) 표시 영역의 최대 값을 나타내는 숫자 또는 레퍼런스(예 : 6000). 기본값은 합계의 최대 값 (점의 합계 값)입니다.

colorPositive: (선택 사항) 첫 번째 또는 마지막 양수인 스파크 라인 상자의 색상을 나타내는 문자열입니다 (이 점의 값은 양수입니다). 기본값은 "# 8CBF64"입니다.

colorNegative: (선택 사항) 첫 번째 또는 마지막 음수인 스파크 라인 상자의 색상을 나타내는 문자열입니다 (이 점의 값은 음수입니다). 기본값은 "# D6604D"입니다.)

vertical: (선택 사항) 상자의 방향이 세로 또는 가로인지 여부를 나타내는 Boolean입니다. 기본값은 false입니다.
