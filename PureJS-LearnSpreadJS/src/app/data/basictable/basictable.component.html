기본 테이블 | Basic Table

1. 때로는 관련 데이터 그룹을 보다 쉽게 관리하고 분석 할 수 있기를 원합니다. Spread.Sheets은 당신을 도울 테이블을 제공합니다. 셀 범위를 테이블로 변환 할 수 있습니다.

2. 두 가지 다른 방법으로 테이블을 추가 할 수 있습니다. 첫 번째 방법은 다음 코드와 같이 add 메서드를 사용하여 빈 테이블을 추가하는 것입니다. 

var spread = $('workbook');
var  sheet = spread.getActiveSheet();
var  tableStyle = GC.Spread.Sheets.Tables.TableThemes.light1;
var  table = sheet.tables.add('table1', 0, 0, 4, 4, tableStyle);

두 번째 방법은 테이블을 데이터 소스로 추가하고 addFromDataSource를 사용하는 것입니다. 예 : 

var  source = [
{ LastName: 'Jone', FirstName: 'Nancy',Phone: '(123)555-0100' },
{ LastName: 'Tom', FirstName: 'Andrew',Phone: '(123)555-0100' },
{ LastName: 'Kotas', FirstName: 'Jan',Phone: '(123)555-0100' },
var  table = sheet.tables.addFromDataSource( 'table2 , 1, 1 , source);

두 번째 방법을 사용하면 테이블의 일부 데이터를 초기화 할 수 있습니다.
