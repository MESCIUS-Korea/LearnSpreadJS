코멘트 동작 | Comment Actions

1. 코멘트은 끌어 놓기, 크기 조정, 이동 등과 같은 작업을 지원합니다.

2. 다음 키는 코멘트 작업을 위한 조치를 제공합니다.

  • Esc : 코멘트의 편집 상태에 있는 경우 Esc 키를 누르면 코멘트이 활성 상태가됩니다. Esc 키를 다시 누르면 코멘트의 선택이 취소됩니다.
  • Delete: 코멘트을 선택하고 상태가 활성화되면 Delete 키를 누르면 선택한 코멘트이 삭제됩니다.
  • Up: 댓글을 선택하고 상태가 활성화 된 경우 위쪽 화살표를 눌러 댓글을 위로 이동합니다.
  • Down: 설명을 선택하고 상태가 활성화 된 경우 아래쪽 화살표를 눌러 코멘트을 아래로 이동합니다.
  • Left: 댓글을 선택하고 상태가 활성화 된 경우 왼쪽 화살표를 눌러 댓글을 왼쪽으로 이동합니다.
  • Right: 메모를 선택하고 상태가 활성화 된 경우 오른쪽 화살표를 눌러 메모를 오른쪽으로 이동하십시오.

3. 마우스 동작을 사용하여 코멘트과 상호 작용할 수도 있습니다. 코멘트을 선택하면 위치 표시기와 8 개의 크기 표시기가 표시됩니다. 크기 조정 표시기를 드래그하여 설명의 크기를 조정할 수 있습니다. 또한 코멘트을 드래그하여 다른 위치에 놓을 수도 있습니다.
