기본 스파크라인 | Basic Sparkline

1. 스파크 라인은 시트 셀에있는 작은 차트로 데이터를 시각적으로 나타냅니다. 스파크 라인을 사용하여 계절별 증가 또는 감소, 경기 순환 등과 같은 일련의 값의 추세를 보여줍니다.

2. 스파크 라인을 만들려면 setSparkline 메서드를 사용하여 셀의 스파크 라인을 설정합니다. getSparkline 메서드를 사용하여 스파크 라인을 가져올 수 있습니다. 예 : 

var spread = $('workbook');
var sheet = spread.getActiveSheet();
var data = new GC.Spread.Sheets.Range(1, 0, 8, 1);
var dateAxis = new GC.Spread.Sheets.Range(1, 1, 8, 1);
sheet.addSpan(11, 0, 4, 3);
sheet.setSparkline(11, 0, data
   , GC.Spread.Sheets.Sparklines.DataOrientation.vertical
   , GC.Spread.Sheets.Sparklines.SparklineType.line
   , setting
   , dateAxis
   , GC.Spread.Sheets.Sparklines.DataOrientation.vertical
);
sheet.getSparkline(11, 0);

스파크 라인에는 세 가지 유형이 있습니다. 열거 형 SparklineType은 스파크 라인 유형을 나타냅니다.

  • line
  • column
  • winloss

3. 스파크 라인을 제거하려면 removeSparkline 메서드를 사용하여 지정된 셀의 스파크 라인을 제거합니다. 예 : 

var spread = $('workbook');
var sheet = spread.getActiveSheet();
sheet.removeSparkline(11, 0);
