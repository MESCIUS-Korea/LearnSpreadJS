위치 고정 | Fixed Position

1. fixedPosition 메서드를 사용하여 플로팅 객체를 고정 된 위치에 표시할지 여부를 설정할 수 있습니다. 

picture.fixedPosition (true);

2. 플로팅 객체 (그림 또는 FloatingObject)의 fixedPosition이 true 일 때 행 또는 열을 스크롤하거나 숨기거나 표시하거나, 크기 조정 또는 이동 한 후에는 플로팅 객체의 위치가 변경되지 않습니다.

3. fixedPosition이 true이면 dynamicMove 및 dynamicSize가 무시됩니다.

4. fixedPosition이 true이면 플로팅 객체를 이동하거나 크기를 조정할 수 있습니다.

5. fixedPosition이 true이면 다음 속성이 이전처럼 작동합니다.

  • position
  • height
  • width
  • startRow
  • startRowOffset
  • startColumn
  • startColumnOffset
  • endRow
  • endRowOffset
  • endColumn
  • endColumnOffset

샘플코드:

결과화면과 함께 HTML, JS 코드도 확인이 가능합니다.
