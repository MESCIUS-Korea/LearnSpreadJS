Scatter 스파크라인 | Scatter Sparkline

ScatterSparkline 함수를 사용하여 scatter sparkline 함수를 만들 수 있습니다.

=SCATTERSPARKLINE(points1, points2?, minX?, maxX?, minY?, maxY?, hLine?, vLine?, xMinZone?, xMaxZone?, yMinZone?, yMaxZone?, tags?, drawSymbol?, drawLines?, color1?, color2?, dash?)

이 함수에는 다음과 같은 매개 변수가 있습니다.

points1 : x, y 데이터의 첫 번째 Series입니다. "H1 : I3"와 같은 범위입니다. 행 계수가 열 계수보다 크거나 같으면 처음 두 열에서 데이터를 가져옵니다. 첫 번째 열은 x 값을 포함하고 두 번째 열은 y 값을 포함합니다. 그렇지 않으면 처음 두 행에서 데이터를 가져옵니다. 첫 번째 행에는 x 값이 포함되고 두 번째 행에는 y 값이 포함됩니다.

points2 : (선택 사항) x, y 데이터의 두 번째 Series입니다. "H4 : I6"과 같은 범위입니다. 행 계수가 열 계수보다 크거나 같으면 처음 두 열에서 데이터를 가져옵니다. 첫 번째 열은 x 값을 포함하고 두 번째 열은 y 값을 포함합니다. 그렇지 않으면 처음 두 행에서 데이터를 가져옵니다. 첫 번째 행에는 x 값이 포함되고 두 번째 행에는 y 값이 포함됩니다.

minX : (선택 사항) x 두 Series는의 최소 한도, 각 Series는 생략되면 고유 한 값을 가집니다.

maxX : (선택 사항) x 두 Series는의 최대 한도; 각 Series는 생략되면 고유 한 값을 가집니다.

minY : (선택 사항) y 두 Series는의 최소 한도. 각 Series는 생략되면 고유 한 값을 가집니다.

maxY : (선택 사항) y 두 Series는의 최대 한도. 각 Series는 생략되면 고유 한 값을 가집니다.

hLine : (선택 사항) 가로 축의 위치입니다. 생략하면 선이 안 만들어집니다.

vLine : (선택 사항) 세로 축의 위치입니다. 생략하면 선이 안 만들어집니다.

xMinZone : (선택 사항) x 회색 영역의 최소값. 이 네 개의 영역 매개 변수 중 하나라도 생략되면 회색 영역이 없습니다.

xMaxZone : (선택 사항) x 회색 영역의 최대 값. 이 네 개의 영역 매개 변수 중 하나라도 생략되면 회색 영역이 없습니다.

yMinZone : (선택 사항) y 회색 영역의 최소값. 이 네 개의 영역 매개 변수 중 하나라도 생략되면 회색 영역이 없습니다.

yMaxZone : (선택 사항) y 회색 영역의 최대 값. 이 네 개의 영역 매개 변수 중 하나라도 생략되면 회색 영역이 없습니다.

tags : (선택 사항) true이면 y 값이 첫 번째 Serie의 최대 값 인 지점을 "# 0000FF"로 표시하고 y 값이 첫 번째 시리즈의 최소값 인 지점을 "# CB0000"으로 표시합니다. . 기본값은 false입니다.

drawSymbol : (선택 사항) True이면 각 점을 심볼로 그립니다. 첫 번째 Serie의 심볼은 원이고 두 번째 Serie의 심볼은 사각형입니다. 기본값은 true입니다.

drawLines : (선택 사항) True이면 각 Serie의 순서에 따라 각 점을 한 행으로 연결하십시오. 기본값은 false입니다.

color1 : (선택 사항) 첫 번째 일련의 점의 색상 문자열. 기본값은 "# 969696"입니다.

color2 : (선택 사항) 두 번째 일련의 점의 색상 문자열. 기본값은 "# CB0000"입니다.

dash : (선택 사항) True이면 선은 점선입니다. 그렇지 않으면 라인은 실선입니다. 기본값은 false입니다.
