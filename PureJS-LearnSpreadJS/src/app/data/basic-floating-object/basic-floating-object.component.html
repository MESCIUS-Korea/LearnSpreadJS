기본 플로팅 객체 | Basic Floating Object

1. Spread.Sheets에서 플로팅 대상을 시트에 추가 할 수 있습니다. 플로팅 객체는 Cell의 맨 위에 표시됩니다.

2. 플로팅 객체에는 FloatingObject와 Picture의 두 가지 종류가 있습니다.

  • FloatingObject : 사용자가 HTMLElement 내용을 사용자 정의할 수 있도록 content 속성을 표시합니다.
  • Picture : Excel 그림 기능과 유사한 기본 그림 기능을 제공하는 일부 기본 그림 속성을 노출합니다.

3. FloatingObject를 시트에 추가하려면 FloatingObject 객체를 만들고 add 메서드를 사용하여 시트에 추가해야합니다. 예 : 

var customFloatingObject = new GC.Spread.Sheets.FloatingObjects.FloatingObject('f1');
customFloatingObject.startColumn(1);
customFloatingObject.endColumn(6);
customFloatingObject.endRow(6);
var div = document.createElement('div');
div.innerHTML = '<span>Spread.Sheets support FloatingObject</span>' + 
<div style='border: 1px dotted red; width: 80%; margin:auto;'>' + 
'<ul><li>I am list one.</li><li>I am list two.</li><li>I am list three.</li></ul></div>';
customFloatingObject.content(div);
sheet.floatingObjects.add(customFloatingObject);

사용자 정의 플로팅 객체를 추가한 후에는 이 객체를 사용하여 작업 할 수 있습니다. get 메서드를 사용하여 이름을 가져오거나 all 메서드를 사용하여 모든 FloatingObject 객체를 가져옵니다. 필요가 없으면 remove 메소드를 사용하여 이름으로 제거 할 수 있습니다.

4. 시트에 그림을 추가하려면 add 메서드를 사용하여 그림을 시트에 추가 할 수 있습니다. 예 : 

sheet.pictures.add('f2', 'images/splogo.png', 62, 140, 180, 200);
var pic = sheet.pictures.get('f2');
sheet.pictures.remove('f2');

그림을 추가 한 후 그림을 사용하여 작업 할 수 있습니다. get 메서드를 사용하여 이름으로 가져 오거나 all 메서드를 사용하여 모든 Picture 객체를 가져옵니다. 필요가 없으면 remove 메소드를 사용하여 이름으로 제거 할 수 있습니다.

5. FloatingObject와 Picture는 모두 플로팅 객체입니다. 그것들은 Cell의 맨 위에 있지만 다른 객체의 위에있는 Object가 있는 경우, 위의 객체 위에 Object를 올려 놓는 것이 좋습니다. zIndex 메서드를 사용하여 플로팅 객체의 Z- 인덱스를 설정합니다. 예: 

sheet.floatingObjects.zIndex('f1', 2000);
sheet.pictures.zIndex('f2', 2001);
