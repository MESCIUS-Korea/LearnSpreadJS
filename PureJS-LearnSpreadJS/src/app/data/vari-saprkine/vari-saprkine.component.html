Vari 스파크라인 | Vari Sparkline

VariSparkline 함수를 사용하여 Vari 스파크 라인을 만들 수 있습니다.

=VARISPARKLINE(variance, reference?, mini?, maxi?, mark?, tickunit?, legend?, colorPositive?, colorNegative?, vertical?) 이 함수에는 다음과 같은 매개 변수가 있습니다.

variance : 막대의 길이를 나타내는 숫자 또는 참조 (예 : 2 또는 "A1").

reference : (선택 사항) 레퍼런스 선의 위치를 ​​나타내는 숫자 또는 레퍼런스 (예 : 0 또는 "A2"). 기본값은 0입니다.

mini: (선택 사항) 스파크 라인의 최소값을 나타내는 숫자 또는 레퍼런스 (예 : -5 또는 "A3"). 기본값은 -1입니다.

maxi: (선택 사항) 스파크 라인의 최대 값을 나타내는 숫자 또는 레퍼런스 (예 : 5 또는 "A4"). 기본값은 1입니다.

mark: (선택 사항) 마크 라인의 위치를 ​​나타내는 숫자 또는 레퍼런스 (예 : 3 또는 "A5"). 기본값은 0입니다.

tickunit : (선택 사항) 눈금 단위를 나타내는 숫자 또는 레퍼런스 (예 : 1 또는 "A6"). 기본값은 0입니다.

legend : (선택 사항) 텍스트를 표시할지 여부를 나타내는 Boolean입니다. 기본값은 false입니다.

colorPositive : (선택 사항) 분산이 레퍼런스보다 클 경우의 색 구성표를 나타내는 문자열. 기본값은 "녹색"입니다.

colorNegative : (선택 사항) 분산이 레퍼런스보다 작은 경우의 색 구성표를 나타내는 문자열. 기본값은 "빨간색"입니다.

vertical : (선택 사항) 상자의 방향이 세로 또는 가로인지 여부를 나타내는 Boolean입니다. 기본값은 false입니다.
