셀-레벨 바인딩 | Cell-Level Binding

1. 데이터 소스를 셀에 바인딩 할 수 있습니다. Spread.Sheets는 셀 수준의 바인딩을 지원합니다.

2. 셀 바인딩 소스를 만든 다음 setBindingPath 메서드를 사용하여 지정된 시트 영역의 지정된 셀에서 셀 수준 바인딩의 바인딩 경로를 설정할 수 있습니다. 그런 다음 시트의 데이터 소스를 설정하십시오. 예 : 

var person = { name: 'Wang feng', age : 25, sex: 'male',  address: { postcode: '71075' }}
var source = new  GC.Spread.Sheets.Bindings.CellBindingSource(person);
sheet.setBindingPath(0, 0, 'name');
sheet.setBindingPath(1, 1, 'age');
sheet.setBindingPath(3, 3, 'sex');
sheet.setBindingPath(4, 4, 'address.postcode');
sheet.setDataSource(source);

getBindingPath를 사용하여 지정된 시트 영역의 지정된 셀에서 셀 레벨 바인딩의 바인딩 경로를 가져올 수 있습니다.

결과화면과 함께 HTML, JS 코드도 확인이 가능합니다.
