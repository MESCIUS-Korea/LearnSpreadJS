검색 태그 | Search Tag

문자열 유형 인 셀 태그를 검색 할 수 있습니다. 검색 순서는 zOrder (열 우선) 또는 nOrder (행 우선)로 설정할 수 있습니다.

1. SearchCondition을 준비한 다음 검색 태그에 대해 searchTarget을 SearchFoundFlags.cellTag로 설정합니다.

2. sheet.search (searchCondition)를 사용하여 검색을 수행하십시오.

검색 결과의 findRowIndex 및 findColumnIndex는 첫 번째로 일치하는 셀 위치를 알려줍니다. 둘 다 찾아지지 않으면 -1이됩니다. 

var spreadNS = GC.Spread.Sheets;
var condition = new spreadNS.Search.SearchCondition();
condition.searchTarget = spreadNS.Search.SearchFoundFlags.cellTag;
condition.searchString = 'demo';
condition.searchOrder = spreadNS.Search.SearchOrder.zOrder;
// ... 
var result = sheet.search(condition);
if (result.foundRowIndex < 0 && result.foundColumnIndex < 0) {
// ...
