사용자 정의 스파크라인 | Custome Sparkline

1. 스파크 라인을 작성한 후 제공된 API를 사용하여 스파크 라인 그룹 설정, 스파크 라인 유형 변경, 스파크 라인 데이터 설정, 날짜 축 데이터 설정 등의 스파크 라인을 사용자 정의 할 수 있습니다.

2. 스파크 라인을 그룹화 할 수 있습니다. 그 후에 이 스파크 라인은 몇 가지 공통 설정을 공유합니다. groupSparkline 및 ungroupSparkline 메소드를 사용하여 스파크 라인을 그룹화하거나 그룹 해제하십시오. 예 : 

var spread = $('#ss').data('workbook');
var sheet = spread.getActiveSheet();
vares = sheet.getSparkline(11, 0);
var esg = sheet.groupSparkline([es]);
esg.sparklineType = GC.Spread.Sheets.Sparklines.SparklineType.column;
sheet.ungroupSparkline(esg);

3. sparklineType 메소드를 사용하여 sparkline 유형을 변경할 수 있습니다. 예 : 

var sparkline = sheet.getSparkline(11, 0);
sparkline.sparklineType(GC.Spread.Sheets.Sparklines.SparklineType.column);

4. 또한 data 및 dataAxisData 메소드를 가져오고 설정할 수 있습니다. data 메소드를 사용하여 데이터 오브젝트를 가져오고 설정하십시오. dataOrientation 메서드를 사용하여 데이터 방향을 가져오고 설정합니다. dateAxisData 메서드를 사용하여 날짜 축 데이터 객체를 가져오고 설정합니다. dateAxisOrientation 메서드를 사용하여 날짜 축 방향을 가져오고 설정합니다. displayDateAxis 메서드를 사용하여 날짜 축 표시 여부를 가져오고 설정합니다. 예 : 

var sparkline = sheet.getSparkline(11, 0);
sparkline.data(new GC.Spread.Sheets.Range(1, 0, 8, 1));
sparkline.dataOrientation(GC.Spread.Sheets.Sparklines.DataOrientation.vertical);
sparkline.dateAxisData(new GC.Spread.Sheets.Range(1, 2, 8, 1));
sparkline.dateAxisOrientation(GC.Spread.Sheets.Sparklines.DataOrientation.vertical);
sparkline.displayDateAxis(true);

5. 스파크 라인 설정이 많이 있습니다. setting 메소드를 사용하여 스파크 라인 설정을 가져오고 설정할 수 있습니다. 다음 단계에서 설정에 대한 자세한 내용을 볼 수 있습니다. 예 : 

var sparkline = sheet.getSparkline(11, 0);
var setting  = new GC.Spread.Sheets.Sparklines.SparklineSetting();
setting.showHigh(true);
sparkline.setting(setting);

6. 스파크 라인 이벤트가 있습니다. 스파크 라인이 변경되면 발생합니다. 예 : 

sheet.bind(GC.Spread.Sheets.Events.SparklineChanged, function() {
//event handler 
});
