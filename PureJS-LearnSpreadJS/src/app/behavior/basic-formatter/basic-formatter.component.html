기본 포멧터 | Basic Formatter

1. Spread.Sheets에서 각 셀의 형식을 사용자 정의하여 원하는 형식으로 데이터를 표시 할 수 있습니다.

2. 셀의 포맷터를 설정하려면 setFormatter 메소드를 사용하여 포맷터 문자열을 셀에 설정하고 getFormatter 메소드를 사용하여 셀의 포맷터를 가져옵니다. 또한 formatter 메소드를 사용하여 셀, 열 또는 행의 설정을 가져와 다른 셀의 포맷터로 설정할수 있습니다. 예를들면: 

sheet.setValue(9, 6, new Date(2013,3,1));
sheet.setFormatter(9, 6, 'd-mmm;@');
//or
sheet.getCell(9, 6).formatter('d-mmm;@');

3. 셀의 일반 형식을 설정할 수 있습니다. GeneralFormatter를 만들어 셀에 설정합니다. 예를들면: 

var generalFormat = new GC.Spread.Formatter.GeneralFormatter(
            '[Red]+0.00;[Green]-0.00;[Yellow]0;[Blue]@');
var cell = sheet.getCell(0, 0);
cell.formatter(generalFormat);

결과화면과 함께 HTML, JS 코드도 확인이 가능합니다.
