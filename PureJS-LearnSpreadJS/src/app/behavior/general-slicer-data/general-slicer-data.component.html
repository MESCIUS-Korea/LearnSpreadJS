일반 슬라이서 데이터 | General Slicer Data

GeneralSlicerData는 2 차원 배열의 데이터를 처리하는 데 사용됩니다. 하나의 GeneralSlicerData에 여러 개의 슬라이서를 부착 할 수 있으며 각 슬라이서는 하나의 데이터 열을 필터링하는 데 사용됩니다. 모든 슬라이서가 필터링되면, 모든 슬라이서는 GeneralSlicerData로부터 통지를 받습니다. 동시에 모든 슬라이서는 GeneralSlicerData에서 필터링 된 결과를 가져와 UI를 업데이트합니다.

슬라이서는 다음 단계에 따라 GeneralSlicerData와 함께 작동합니다.

(1) 데이터로 새 GeneralSlicerData를 만듭니다.

(2) 새 사용자 정의 슬라이서를 작성하고 GeneralSlicerData에 연결하십시오.

(3) GeneralSlicerData에서 열 데이터를 가져 와서 슬라이서 UI를 만듭니다.

(4) UI 이벤트에 응답하고 GeneralSlicerData doFilter 메소드를 호출하십시오.

(5) GeneralSlicerData에서 필터링 된 결과를 가져 와서 슬라이서 UI를 업데이트합니다.

다음 API는 간단한 코드를 작성하는 데 도움이 됩니다.

getData : 지정된 열에 대한 모든 데이터를 가져옵니다.

getExclusiveData : 지정된 열에 대한 독점 데이터 (비 반복 데이터)를 가져옵니다.

doFilter : 지정된 열 및 배타적 데이터 인덱스에 해당하는 데이터를 필터링합니다.

doUnfilter : 지정된 열에 해당하는 데이터의 필터를 해제합니다.

attachListener : 슬라이서 데이터에 슬라이서를 연결합니다.

detachListener : 슬라이서 데이터에서 슬라이서를 분리합니다.

onFiltered : 슬라이서 데이터가 필터링 된 후에 발생합니다.

샘플코드:

결과화면과 함께 HTML, JS 코드도 확인이 가능합니다.
