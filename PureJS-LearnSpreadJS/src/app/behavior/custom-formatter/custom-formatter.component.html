사용자 정의 포멧터 | Custom Formatter

FormatterBase를 사용하면 셀 텍스트의 서식 지정 및 색상 지정 규칙을 사용하여 사용자 정의 포맷터를 만들 수 있습니다.

1. FormatterBase에 클래스 기반을 만듭니다. 예를들면 : 

function CustomNumberFormat() {
}
CustomNumberFormat.prototype = new GC.Spread.Formatter.FormatterBase();

2. 셀 텍스트의 형식 및 색상에 대한 규칙을 정의하는 형식 (obj, conditionalForeColor)을 아래와 같이 제공합니다.

  • obj : 원시 셀 값입니다.
  • conditionalForeColor : 텍스트의 색상값입니다(선택 사항). 값을 색상 문자열로 설정하여 규칙을 적용할수 있습니다.

format 메소드는 형식화 된 문자열을 반환합니다. 동시에 값 범위를 기준으로 텍스트에 색을 지정할 수 있습니다. 예를들면 : 

function formatNumber(value, conditionalForeColor) { 
   if (isFinite(value)) { 
       if (value >= 10) { 
           conditionalForeColor.value = "green"; 
       } else if (value > 0) { 
           conditionalForeColor.value = "gold"; 
       } else { 
           conditionalForeColor.value = "blue"; 
       }
   } else { 
       conditionalForeColor.value = "red"; 
   }
   return value.toString(); 
}

3. parse (str)를 사용하여 지정된 문자열에서 파싱 된 값을 반환합니다.

결과화면과 함께 HTML, JS 코드도 확인이 가능합니다.
