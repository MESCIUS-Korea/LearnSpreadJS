기본 조건부 서식 | Basic Conditional Format

1. 수천 행의 데이터가 있는 시트의 경우 개개의 데이터를 통해 전체적인 패턴이나 추세를 이해하는 것은 매우 어렵습니다. 스파크 라인과 마찬가지로 조건부 서식은 데이터를 시각화하고 시트를 더 쉽게 이해할 수있는 또 다른 방법을 제공합니다.

2. Spread.Sheets에 조건부 서식을 추가하려면 먼저 conditionalFormats를 사용하여 시트의 조건부 서식을 가져옵니다. 그런 다음 조건부 규칙을 만들고 addRule 메서드를 사용하여 추가 할 수 있습니다. 또한 addSpecificTextRule, addCellValueRule 등과 같이 지정된 규칙의 add 메소드를 사용할 수도 있습니다. 예를들면 : 

var style = new GC.Spread.Sheets.Style();
style.backColor = '#CCFFCC';
var cvRule = cfs.addCellValueRule(GC.Spread.Sheets.ConditionalFormatting.ComparisonOperators.greaterThan, 100,0, style, [new GC.Spread.Sheets.Range(3, 3, 3, 3)]);

3. 시트에 일부 규칙을 추가 한 후에는 getRule 메소드를 사용하여 규칙 색인이나 또는 모든 조건부 규칙을 가져올 수 있습니다. 예를들면: 

var cfs = sheet.conditionalFormats;
// 새로 추가 된 것이 끝에 있습니다.
var cvRule = cfs.getRule(1);

// 또는 모든 조건부 규칙을 가져갑니다.
var rules = cfs.getRules();

4. 조건부 규칙을 제거하려면 removeRule, removeRuleByRange 또는 clearRule 메서드를 사용합니다. 예를들면 : 

var cfs = sheet.conditionalFormats;
cfs.removeRule(cvRule)
cfs.removeRuleByRange(3, 3, 3, 3);
cfs.clearRule(); //모든 규칙 삭제
