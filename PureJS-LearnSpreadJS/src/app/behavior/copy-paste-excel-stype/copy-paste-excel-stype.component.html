스타일 복사/붙여넣기 | Copy Paste Excel Style

1. Spread.Sheets는 Excel에서 복사 스타일 / Span을 지원하고 Spread.Sheets Sheets에 붙여넣고 Spread.Sheets Sheets에서 스타일 / Span을 복사 한 다음 Excel에 붙여 넣습니다.

2.이 기능을 사용하려면 allowCopyPasteExcelStyle이 true 일 때 통합 문서 옵션을 설정하십시오. 예 : 

workbook.options.allowCopyPasteExcelStyle = true;

그렇다면 Excel에서 스타일을 복사하여 Excel로 복사 할 수 있으며 기본값은 true입니다.

이제 ClipboardChanging 및 ClipboardChanged에서 이벤트 인수의 copyData가 string에서 Object {text, html}로 변경되었습니다.

ClipboardPasting 및 ClipboardPasted에서 pasteData {text, html}가 이벤트 파라미터에 추가되었습니다.

샘플코드:

결과화면과 함께 HTML, JS 코드도 확인이 가능합니다.
