버튼 셀 타입 | Button Cell Type

1. 버튼은 버튼 셀을 나타냅니다.

2. 버튼 셀을 만들려면 다음 예제대로 하시면 됩니다. 

var b1 = new GC.Spread.Sheets.CellTypes.Button();
sheet.setCellType(3, 2, b1, GC.Spread.Sheets.SheetArea.viewport);

3. 제공되는 많은 API를 사용하여 Button을 사용자 정의 할 수 있습니다. 셀의 버튼 위치를 제어하려면 다음 방법을 사용하십시오.

  • marginTop : 셀을 기준으로 버튼의 위쪽 여백을 픽셀 단위로 가져 오거나 설정합니다.
  • marginRight : 셀을 기준으로 버튼의 오른쪽 여백을 픽셀 단위로 가져 오거나 설정합니다.
  • marginBottom : 셀을 기준으로 버튼의 아래쪽 여백을 픽셀 단위로 가져 오거나 설정합니다.
  • marginLeft : 셀을 기준으로 버튼의 왼쪽 여백을 픽셀 단위로 가져 오거나 설정합니다. 
b1.marginLeft(15);
b1.marginTop(7);
b1.marginRight(15);
b1.marginBottom(7);

4. text 메서드를 사용하여 버튼의 내용을 가져오고 설정할 수 있습니다. buttonBackColor 메서드를 사용하여 버튼의 배경색을 가져오고 설정합니다. 예 : 

b1.text('Margin');
b1.buttonBackColor('red');
