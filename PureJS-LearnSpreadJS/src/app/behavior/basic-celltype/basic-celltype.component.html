기본 셀 타입 | Basic Cell Type

1. Spread.Sheets은 셀 유형을 제공합니다. 양방향 바인딩을 지원할 수 있습니다 (뷰 모델의 데이터 변경 사항이 뷰에 반영됩니다). 각 셀 유형은 디스플레이 모드와 편집 모드의 두 가지 모드를 제공합니다.

2. 셀 유형을 설정하려면 먼저 cellType 객체를 만든 다음 setCellType 메서드를 사용하여 시트, 셀, 열 또는 행의 cellType을 설정합니다. 예 : 

var  b1 = new  GC.Spread.Sheets.CellTypes.Button();
b1.text('Margin Left');
b1.marginLeft(40);
sheet.setCellType(3, 2, b1, GC.Spread.Sheets.SheetArea.viewport);
//or
sheet.getCell(3,2).cellType(b1);

경우에 따라 cellType을 열에 바인딩하여 데이터 바인딩과 함께 cellType을 사용할 수 있습니다. 예 : 

var lines = ['Computers', 'Washers', 'Stoves'];
var lineCellType = new GC.Spread.Sheets.CellTypes.ComboBox();
lineCellType.items(lines);
var columns = [
   {name: 'name', size: 100}
   {name: 'color', cellType: lineCellType, size: 80},
];
sheet.bindColumns(columns)
b1.marginLeft(40);

셀 유형을 설정 한 후 getCellType 메소드를 사용하여 셀 유형을 가져올 수 있습니다. 셀 유형을 제거하려면 값을 null 또는 undefined로 설정하십시오.

3. Spread.Sheets는 스프레드 시트에서 이벤트 (ButtonClicked)를 제공합니다. 이 이벤트는 셀의 단추, 확인란 또는 하이퍼 링크를 클릭 할 때 발생합니다. 예 : 

spread.bind(GC.Spread.Sheets.Events.ButtonClicked, function() {
   //do some thing...
});
