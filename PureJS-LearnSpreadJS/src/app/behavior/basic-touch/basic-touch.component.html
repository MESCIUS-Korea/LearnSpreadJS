기본 터치 | Basic Touch

1. 스프레드 시트는 터치를 지원합니다.

2. 스프레드 시트는 다음 플랫폼에서 터치를 지원합니다.

  • 데스크톱 : Windows - IE 9 이상, Chrome, Firefox, Opera
  • 태블릿 PC : iPad - Chrome, Safari; Android pad - Chrome; Surface--IE 10이상
  • 모바일 : Android - Chrome; Windows Phone : IE 10 이상

3. useTouchLayout을 사용하면 터치 레이아웃을 사용하여 스프레드 위젯을 표시할지 여부를 가져오고 설정할 수 있습니다. 터치 레이아웃에서 컨트롤을 손쉽게 터치로 조작 할 수 있습니다. 

spread.options.useTouchLayout = true;

4. 스프레드 시트는 터치 모드에서 스크롤, 확대 / 축소, 선택, 크기 조정, 드래그 채우기, 그룹, cellType, 자동 맞춤, 필터, 편집, tabStrip 및 toolStrip등 많은 기능을 지원합니다.

샘플코드:

결과화면과 함께 HTML, JS 코드도 확인이 가능합니다.
