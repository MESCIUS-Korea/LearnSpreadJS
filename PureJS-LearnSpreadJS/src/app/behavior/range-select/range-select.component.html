범위 선택 | Range Select

Spread.Sheets에서 범위를 수동으로 입력하는 대신 범위를 선택하여 최종 사용자가 범위 (예 : "Sheet1! $ A $ 5 : $ C $ 10")를 입력하도록 허용해야하는 경우가 있습니다. 이 기능의 결과 또는 반환 값은 텍스트 상자에있는 하나 이상의 범위의 주소를 나타내는 텍스트입니다 (두 개의 범위 "Sheet1! A1 : B5, Sheet2! G11"의 예).

1. 범위 선택 모드를 사용하려면 {rangeSelectMode : true} 옵션이 있는 formulaTextBox를 만들어야 합니다. 예 : 

var rangeSelect = createElement("div");
var fbx = new GC.Spread.Sheets.FormulaTextBox.FormulaTextBox(rangeSelect, {rangeSelectMode: true});

2. FormulaTextBox의 통합 문서 메서드를 사용하여이 formulaTextBox를 스프레드에 첨부 할 수 있습니다. 예를 들면 다음과 같습니다. 

var  workbook = new  GC.Spread.Sheets.workbook(document.getElementById('ss'))
fbx.workbook(workbook);

fbx1.workbook ()에 인수가 없으면 범위 분리 선택

3. FormulaTextBox의 텍스트 메소드를 사용하여 수식 텍스트 상자의 텍스트를 가져올 수 있습니다. 예 : 

var str = fbx.text();

4. destroy 메소드를 사용하여 수식 텍스트 상자의 모든 요소를 제거 할 수 있습니다. 예 :

fbx.destroy();

샘플코드:

결과화면과 함께 HTML, JS 코드도 확인이 가능합니다.
