사용자 정의 유효성 검사 | Custom Data Validators

1. 제목, 입력 메시지, 오류 메시지 등과 같은 유효성 검사를 사용자 정의 할 수 있습니다. 이 설정을 사용하면 셀에 입력 할 수 있거나 입력해야하는 데이터를 알 수 있으며 데이터가 유효하지 않으면 에러가 표시됩니다.

2. 유효성 검사 개체를 사용할 때 해당 속성을 설정할 수 있습니다. 예를들면: 

var dv1 = sheet.getDataValidator(1,1);
dv1.inputTitle('Please choose a category:');
dv1.inputMessage('Fruit, Vegetable, Food');
dv1.showInputMessage(true); //제목과 메세지가 출력여부를 결정합니다.
dv1.showErrorMessage(true); //에러 메세지가 출력여부를 결정합니다.
dv1.inCellDropdown(true); //드롭 다운 버튼 표시 여부를 결정합니다.
dv1.ignoreBlank(false); //비어 있는 값을 표시할지 여부를 결정합니다.
dv1.errorStyle(GC.Spread.Sheets.DataValidation.ErrorStyle.stop);

3. 유효하지 않은 데이터를 입력하면 컨트롤이 ValidationError 이벤트를 실행합니다. 예를들면 : 

sheet.bind(GC.Spread.Sheets.Events.ValidationError, function(e, args) {
  // 원하는 코드를 넣으세요.
});
