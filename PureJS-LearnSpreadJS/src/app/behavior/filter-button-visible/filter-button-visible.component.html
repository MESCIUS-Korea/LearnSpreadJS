필터 버튼 보이기 | Filter Button Visible

Filter 또는 Table 객체의 filterButtonVisible 메서드를 사용하여 필터 버튼의 표시 여부를 가져 오거나 설정합니다.

filterButtonVisible 메소드의 get 및 set 데이터는 아래와 같이 인수에 따라 다른 속성을 가집니다.

  • 인수 없음: 필터 버튼의 표시 여부를 가져 옵니다. 보이는 경우 True, 보이지 않는 경우 false로 표시 됩니다.
  • 1개의 인수: 만약 인수가 숫자라면, 해당 숫자에 해당하는 열에 필터가 적용되거나 또는 상태가 리턴 됩니다. 만약 인수가 Boolean(True/False) 값이라면, 전체 열의 영역에서 필터가 적용되거나 해제되게 됩니다.
  • 2개의 인수: 첫 번째 인수는 열의 인덱스이고 두 번째 인수는 Boolean(True/False) 값입니다. 해당 열의 인덱스에 필터 버튼을 표시할지 여부를 설정합니다.

예를 들면: 

// 모든 필터버튼 숨기기
filter.filterButtonVisible(false);
// table.filterButtonVisible(false);

// 모든 필터 버튼 보이기
filter.filterButtonVisible(true);
// table.filterButtonVisible(true);

// 특정 컬럼의 필터 버튼 숨기기
filter.filterButtonVisible(colIndex, false);
// table.filterButtonVisible(tableColumnIndex, false);

// 특정한 컬럼의 필터 버튼 보이기 속성 가져오기
filter.filterButtonVisible(colIndex);
// table.filterButtonVisible(tableColumnIndex);
