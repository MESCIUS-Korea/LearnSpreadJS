사용자 정의 채우기 | Custom Fill

1. 마우스 끌기 외에도 채우기 데이터를 할수 있는 방법은 많이 있습니다.

사용자는 아래와 같이 5 가지 형태로 데이터를 채우기 할 수 있습니다:

  • Auto: 자동완성 유형입니다.
  • Direction: 원하는 방향으로 채우기를 선택할수 있는 유형입니다..
  • Linear: 선형 채우기 유형입니다.
  • Growth: 증가 채우기 유형입니다.
  • Date: 날짜 채우기 유형입니다.

2. Spread.Sheets는 아래와 같이 채우기 기능을 위한 메소드를 제공합니다.. 

// 지정된 범위를 자동으로 채웁니다.
// 값이 문자열이면 값이 다른 셀에 복사됩니다.
// 값이 숫자이면 TREND 수식에 의해 새 값이 생성됩니다.
sheet.fillAuto(startRange, fillRange, {series: GC.Spread.Sheets.Fill.FillSeries.column, fillType: GC.Spread.Sheets.Fill.FillType.auto});

// 지정된 방향으로 지정된 범위를 채 웁니다.
sheet.fillAuto(startRange, fillRange, {direction: GC.Spread.Sheets.Fill.FillDirection.left, fillType: GC.Spread.Sheets.Fill.FillType.direction});

// 소스 값 유형이 number인 경우 지정된 범위 선형 추세를 채 웁니다.
// 다음 값은 step 및 stop 값에 의해 생성됩니다.
// 다음 값은 현재 셀 값에 단계 값을 추가하여 계산됩니다.
sheet.fillAuto(startRange, fillRange, {series: GC.Spread.Sheets.Fill.FillSeries.column, step: step, stop: stop, fillType: GC.Spread.Sheets.Fill.FillType.linear});

// 소스 값 유형이 number 일 때 지정된 범위 증가 추세를 채 웁니다.
// 다음 값은 step 및 stop 값에 의해 생성됩니다.
// 다음 값은 단계 값에 현재 셀을 곱하여 계산됩니다.
sheet.fillAuto(startRange, fillRange, {series: GC.Spread.Sheets.Fill.FillSeries.column, step: step, stop: stop, fillType: GC.Spread.Sheets.Fill.FillType.growth};

// 소스 값 유형이 날짜 일 때 지정된 범위를 채 웁니다.
// 다음 값은 현재 값에 단계 값을 추가하여 생성됩니다.
// 단계 값은 날짜 단위를 기준으로 증가합니다.
sheet.fillAuto(startRange, fillRange, {series: GC.Spread.Sheets.Fill.FillSeries.column, unit: GC.Spread.Sheets.Fill.FillDateUnit.month, step: step, stop: stop, fillType: GC.Spread.Sheets.Fill.FillType.date});
