기본 채우기 | Basic Fill

1. 시트에 수동으로 데이터를 입력하는 대신 자동 채우기 기능을 사용하여 패턴을 따르거나 다른 셀의 데이터를 기반으로하는 데이터로 셀을 채울 수 있습니다.

2. 여러 데이터 유형을 빠르게 채우려면 셀을 선택하고 채우기 핸들을 끌어서 채웁니다. 채우기 핸들을 사용하려면 사용할 셀을 선택한 다음 선택된 셀의 우측 하단에 생겨나는 채우기 핸들을 마우스로 클릭한 상태에서 위나 아래로 드래그합니다.

3. Ctrl 키를 누른 상태에서 두 개 이상의 셀 중에서 선택하여 채우기 핸들을 드래그하여 시리즈 자동 채우기를 적용하지 않을수도 있습니다. 채우기를 드래그하면 자동 채우기 옵션 버튼이 표시됩니다. 단추를 클릭하고 선택 영역 채우기 방법을 변경할 수 있습니다. 예를 들어, [서식만 채우기]를 클릭하여 셀 서식만을 채울 수 있습니다. 제공되는 자동 채우기 옵션은 다음과 같습니다.

  • CopyCells : 셀을 값, 형식 및 수식을 포함한 모든 데이터 개체로 채웁니다.
  • FillSeries : 셀을 시리즈로 채웁니다.
  • FillFormattingOnly : 셀을 서식으로만 채웁니다.
  • FillWithoutFormatting : 셀을 값으로 채우고 서식을 지정하지 않습니다.

또한 시작 범위로 드래그 채우기를 사용하여 채워진 값을 지울 수 있습니다.

4. 채우기 핸들을 끌 때, 기본적으로 Spread.Sheets는 채워질 데이터를 우측에 팁으로 표시합니다. 다음 예제와 같이 팁 디스플레이를 끌 수 있습니다. 

spread.options.showDragFillTip = false;

샘플코드:

결과화면과 함께 HTML, JS 코드도 확인이 가능합니다.
