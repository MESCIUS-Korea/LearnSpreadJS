슬라이서 소개 | Slicer Introduction

슬라이서는 테이블 데이터를 필터링하는 새로운 방법으로 제공됩니다. 필터링 후 테이블에 표시된 데이터를 명확하게 나타냅니다.

1. 슬라이서는 테이블을 사용하기 때문에 슬라이서를 만들려면 먼저 테이블을 추가해야합니다.

다음 코드와 같이 add 메서드를 사용하여 슬라이서를 추가 할 수 있습니다. 

var table = sheet.tables.findByName('table1');
var slicer = sheet.slicers.add('slicer1', 'table1', 'Name');
slicer.position(new GC.Spread.Sheets.Point(40, 180));

2. 슬라이서는 슬라이서 헤더와 슬라이서 바디로 구성됩니다.

Slicer header는 slicer caption과 clear filter 버튼으로 구성되어 있습니다. 슬라이서 캡션은 캡션을 표시하는 데 사용됩니다. 필터 제거 버튼은 슬라이서를 제거하는 데 사용되며 기본적으로 비활성화되어 있습니다. 슬라이서 항목이 필터링 된 경우에만 활성화됩니다.

슬라이서 바디는 슬라이서 항목으로 구성됩니다. 슬라이서 항목에는 4 가지 종류가 있습니다.

  • 데이터 항목 없음 : 항목이 다른 슬라이서에 의해 필터링되었습니다.
  • 데이터 항목 있음 : 항목이 다른 슬라이서에 의해 필터링되지 않았습니다.
  • 선택한 항목 : 항목 자체가 필터링되었습니다.
  • 선택하지 않은 항목 : 항목 자체가 필터링되지 않았습니다.

항목은 다른 슬라이서에 의해 필터링되거나 선택된 항목과 선택되지 않은 항목으로 나뉘어 질 때 데이터 항목 없음 및 데이터 항목 있음으로 나뉘어집니다.
