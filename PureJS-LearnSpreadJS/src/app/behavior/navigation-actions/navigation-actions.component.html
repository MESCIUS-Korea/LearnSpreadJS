네비게이션 동작 | Navigation Actions

1. Spread.Sheets은 SpreadActions에 정의 된 탐색 동작을 지원합니다.

  • moveToNextCell : 다음 셀로 이동합니다 (Tab의 기본 동작).
  • moveToPreviousCell : 이전 셀로 이동합니다 (Shift + Tab의 기본 동작).
  • selectNextControl : 지정되지 않은 경우 spread.nextControl 지정된 요소 또는 자동 감지 된 요소를 선택합니다.
  • selectPreviousControl : 설정되지 않은 경우 spread.previousControl 지정된 요소 또는 자동 감지 된 요소를 선택합니다.
  • moveToNextCellThenControl : Active 셀이 마지막으로 표시된 셀이면 다음 셀로 이동 한 후 다음 컨트롤을 선택합니다.
  • moveToPreviousCellThenControl : Active 셀이 첫 번째 보이는 셀이면 이전 셀로 이동 한 다음 이전 컨트롤을 선택합니다.

2. nextControl 및 previousControl 메서드를 사용하여 선택한 컨트롤을 설정합니다. 예 : 

// 다음 및 이전 컨트롤 설정
spread.nextControl(document.getElementById('myNextControl'));
spread.previousControl(document.getElementById('myPreviousControl'));

// auto detect에 의해 하나를 지정하지 않고 undefined로 설정합니다.
spread.nextControl(undefined);

3. 등록 메소드를 사용하여 네비게이션 키와 해당 조치를 설정하십시오. 예 : 

// 탭을 설정하여 다음 컨트롤 선택
spread.commandManager().register('selectNextControl', GC.Spread.Sheets.Commands.selectNextControl, GC.Spread.Commands.Key.tab, false, false, false, false);
// 이전 컨트롤을 선택하려면 Shift + Tab을 설정하십시오. 
spread.commandManager().register('selectPreviousControl', GC.Spread.Sheets.Commands.selectPreviousControl, GC.Spread.Commands.Key.tab, false, true, false, false);

register의 인수는 다음과 같습니다.

  • name : 명령의 이름.
  • execute : 명령의 실행 기능.
  • key: 키의 유니 코드입니다.
  • ctrl: 액션이 Ctrl 키를 사용하면 True입니다. 그렇지 않으면 false입니다.
  • shift: 액션이 Shift 키를 사용하면 True, 그렇지 않으면 false입니다.
  • alt: 액션이 Alt 키를 사용하면 True, 그렇지 않으면 false입니다.
  • meta: Microsoft Windows의 Macintosh 또는 Windows 키에서 작업이 Command 키를 사용하면 True입니다. 그렇지 않으면 false입니다.

4. Spread가 포커스를 잃을 때 선택을 숨기려면 hideSelection 옵션을 사용하십시오. 예 : 

spread.options.hideSelection = true;
