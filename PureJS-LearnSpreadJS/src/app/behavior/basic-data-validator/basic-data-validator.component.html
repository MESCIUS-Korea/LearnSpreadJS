데이터 유효성 검사 | Basic Data Validator

1. 데이터 유효성 검사는 사용자가 셀에 데이터를 입력할 때 타입이나 크기 등을 제한하는데 사용할 수있는 기능입니다. 사용자가 유효하지 않은 데이터를 입력하지 못하도록 데이터 유효성 검사를 구성 할 수 있습니다.

2. 데이터 유효성 검사를 시트에 추가하려면 유효성 검사를 만든 다음 시트에 추가하면 됩니다. 예를들면: 

var dv1 = new GC.Spread.Sheets.DataValidation.createListValidator('Fruit,Vegetable,Food');
dv1.inputTitle('Please choose a category:');
dv1.inputMessage('Fruit, Vegetable, Food');
sheet.setDataValidator(1,1, dv1);

그래서 다음과 같이 6 가지 유형의 유효성 검사를 만들 수 있습니다.

  • createNumberValidator : 숫자를 기준으로 유효성 검사를 만듭니다.
  • createDateValidator : 날짜를 기반으로 검사를 생성합니다.
  • createTextLengthValidator : 텍스트 길이에 따라 유효성 검사를 생성합니다.
  • createFormulaValidator : 수식을 기반으로 유효성 검사를 만듭니다.
  • createFormulaListValidator : 수식 목록을 기반으로 검사를 만듭니다.
  • createListValidator :리스트에 기반해 유효성 검사를 위한 데이터를 작성합니다.

3. 유효성 검사를 셀에 설정하면 getDataValidator 메서드를 사용하여 셀 데이터 유효성 검사를 가져올 수 있습니다. 그리고 isValid 메서드를 사용하여 셀 값이 유효한지 여부를 확인할 수 있습니다. 예를들면: 

sheet.isValid(1, 1, 'Fruit'); // true
var validList = sheet.getDataValidator(1, 1);
validList.isValid(sheet, 1, 1, 'Fruit')
// 데이터 유효성 검사 초기화
sheet.setDataValidator(1, 1, null);

4. highlightInvalidData 옵션을 사용하여 잘못된 데이터를 강조 표시할지 여부를 가져 오거나 설정할 수 있습니다. 이 값을 true로 설정하면 값이 유효하지 않으면 셀에 빨간색 원이 생깁니다. 

spread.options.highlightInvalidData = true;
