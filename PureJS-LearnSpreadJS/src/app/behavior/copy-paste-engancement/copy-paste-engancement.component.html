복사/붙여넣기 향상 | Copy Paste Enhancement

1. Spread.Sheets는 데이터를 복사하거나 붙여 넣을 때 포함될 헤더를 지정하는 옵션을 제공합니다.

2. Workbook 옵션의 copyPasteHeaderOptions를 설정하여 포함될 헤더를 지정하십시오. 예 : 

workbook.options.copyPasteHeaderOptions = GC.Spread.Sheets.CopyPasteHeaderOptions.allHeaders

GC.Spread.Sheets.CopyPasteHeaderOptions:

  • noHeaders: 0
  • rowHeaders: 1
  • columnHeaders: 2
  • allHeaders: 3

3. 이제 Spread.Sheets는 연속되지 않은 여러 셀을 복사 한 다음 단일 범위를 붙여 넣을 수 있습니다. 그러나 다음과 같이 두 가지 경우 만 지원합니다.

  • 동일한 행 및 행 수
  • 동일한 열 및 열 수

4. 스프레드 시트는 복사시 필터링 된 행을 무시합니다.

샘플코드:

결과화면과 함께 HTML, JS 코드도 확인이 가능합니다.
