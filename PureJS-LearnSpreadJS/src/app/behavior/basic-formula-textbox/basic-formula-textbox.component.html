기본 수식입력줄 | Basic Formula Text Box

1. 수식을 입력 할 때 사용 가능한 함수 이름을 잊어 버릴 수 있습니다. Spread.Sheets은 수식을 입력 할 때 함수 설명을 표시하는 수식 목록을 제공합니다. 수식 목록이 표시되면 Tab 키를 누르면 수식 이름이 자동 완성됩니다.

2. 수식에 범위를 입력하면 범위 표시기가 표시됩니다. 마우스를 클릭하고 범위를 선택할 수 있습니다.

3. Function 클래스를 사용하여 함수를 추가합니다. 예 : 

var fn = new GC.Spread.CalcEngine.Functions.Function('squareFunction', 0, 1, 'Returns x * x.')
fn.evaluate = function (x) {
   return x * x;
};

또한 아래와 같이 함수 설명을 얻을 수 있습니다. 

var functionDescription = fn.description();
