터치 도구 모음 항목 | Touch Tool Strip

1. 특정 영역이 선택된 상태에서 선택 항목을 누르면 터치 도구 모음이 나타납니다.

2. 붙여 넣기, 잘라 내기, 복사 및 자동 완성과 같은 기본 제공 도구 모음 항목을 사용할 수 있습니다.

3. 터치 도구 모음 항목 아이템을 커스터마이징하고 터치 도구 모음 항목 스트립에 새로운 아이템을 추가 할 수도 있습니다 

var text = 'clear'
var image = 'image/clear.jpg';
var item = new GC.Spread.Sheets.Touch.TouchToolStripItem(name, text, image, function () {
   spread.touchToolStrip.close();
   alert('지우기 실행...');
});
spread.touchToolStrip.add(item);

텍스트 메서드를 사용하여 항목의 텍스트를 가져오고 설정하며 글꼴 메서드를 사용하여 항목 텍스트의 글꼴을 가져오고 설정 한 다음 foreColor 메서드를 사용하여 항목 텍스트의 색을 가져오고 설정할 수 있습니다.

4. 터치 도구 모음 항목을 추가 한 후 getItem와 getItems를 사용하여 항목을 가져올 수 있습니다. 또한 모든 항목을 삭제하거나 지울 수 있습니다. 

var foreColor = $("#color").val();
  if (!foreColor) {
      alert("아이템 텍스트의 색상을 입력해 주세요.");
  }
  var name = $("#name").val();
  var item = spread.touchToolStrip.getItem(name);
  if (item) {
      item.foreColor(foreColor);
  }
  var font = $("#font").val();
  if (!font) {
      alert("아이템 텍스트의 폰트를 입력해 주세요.");
  }
  var name = $("#name").val();
  var item = spread.touchToolStrip.getItem(name);
  if (item) {
      item.font(font);
  }

5. 제공된 API를 사용하여 도구 모음을 사용자 정의 할 수 있습니다. 예 : 

var toolStrip = spread.touchToolStrip;
toolStrip.imageAreaHeight(50); //Sets the image area height.
toolStrip.itemHeight(80); //Sets the toolbar item height.
toolStrip.itemWidth(120); //Sets the toolbar item width.
toolStrip.separatorHeight(80); //Sets the toolbar separator height.
spread.touchToolStrip.add(item);
