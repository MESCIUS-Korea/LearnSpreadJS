테이블 슬라이서 데이터 | Table Slicer Data

GeneralSlicerData 외에도 Spread.Sheets은 TableSlicerData도 제공합니다. TableSlicerData의 데이터 소스는 Spread.Sheets SheetTable입니다. TableTable을 사용하려면 TableTable을 TableSlicerData의 생성자로 설정해야합니다. RowFilter에 대한 정보와 SheetTable의 필터 대화 상자에서 확인 된 상태는 TableSlicerData와 연결된 Slicers의 필터 결과와 동기화됩니다. TableSlicerData와 연결된 모든 슬라이서는 onFilter 통지를 받고 코드 또는 필터 대화 상자를 사용하여 SheetTable 필터 이후에 필터링 된 결과를 가져옵니다. 다음과 같이 TableSlicerData를 사용할 수 있습니다. 
// 현재 시트의 테이블 가져 오기
var table = sheet.tables.findByName('table1');
// 테이블에서 TableSlicerData 가져 오기
var slicerData = table.getSlicerData();
//커스텀 슬라이서를 생성하여 dom 트리에 추가하십시오.
var slicer1 = new CustomSlicer($('#cityContainer')[0]);
//열 데이터를 슬라이서로 설정.
slicer1.setData(slicerData, 'City');
