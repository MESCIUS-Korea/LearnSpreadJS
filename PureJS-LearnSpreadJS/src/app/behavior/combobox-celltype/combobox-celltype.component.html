콤보 박스 셀 타입 | Combo Box Cell Type

1. ComboBox는 콤보 박스 셀을 나타냅니다.

2. 콤보 박스 셀을 만들려면 다음 예제대로 하시면 됩니다. 

var combo = new GC.Spread.Sheets.CellTypes.ComboBox();
sheet.setCellType(3, 2, combo, GC.Spread.Sheets.SheetArea.viewport);

3. editorValueType 메소드를 사용하여 기본 데이터 모델에 기록 된 값을 가져오고 설정할 수 있습니다. 에디터 값 유형은 EditorValueType enum입니다.

  • text: 선택한 항목의 텍스트 값을 모델에 씁니다.
  • index: 선택한 항목의 인덱스를 모델에 씁니다.
  • value: 선택한 항목의 해당 데이터 값을 모델에 씁니다. 
combo.editorValueType(GC.Spread.Sheets.CellTypes.EditorValueType.text);

4. 다른 editorValueType 설정은 다른 유형의 에디터 값을 작성합니다. 콤보 상자의 값은 콤보 상자의 드롭 다운 목록 항목에 따라 다릅니다. items 메소드를 사용하여 항목을 가져오고 설정할 수 있습니다. 예 : 

combo.items([
 { text: 'Oranges', value: '11k' },
 { text: 'Apples', value: '15k' },
 { text: 'Grape', value: '100k' }]);

5. 편집 가능한 메서드를 사용하여 사용자가 콤보 박스 에디터에서 입력 할 수 있는지 여부를 설정합니다. 기본값은 false입니다. 선택 만 허용됩니다. 예 : 

editableCombo.editable(true);

6. itemHeight 메서드를 사용하여 드롭 다운 목록에서 각 항목의 높이를 설정할 수 있습니다. 예 : 

combo.itemHeight(24);
