사용자 셀 타입 | Custom Cell Type

1. 자신 만의 custom cellType을 만들어야 할 때가 있습니다. 셀이 표시 모드에있는 경우를 그릴 수 있으며 셀이 편집 모드에 있을 때 편집기를 사용자 정의하고 셀 유형 자체로 마우스와 키보드 상호 작용을 처리 할 수 있습니다.

2. Base를 상속하고 paint, createEditorElement, getEditorValue 등과 같은 필수 메소드를 구현하여 셀 유형을 사용자 정의 할 수 있습니다. 그 후에는 사용자 정의 cellType을 사용할 수 있습니다. 예 : 

function MyCellType() {
MyCellType.prototype = new GC.Spread.Sheets.CellTypes.Base();
MyCellType.prototype.paint = function (ctx, value, x, y, w, h, style, options) {
  //캔버스에 셀을 칠합니다. 
};
MyCellType.prototype.createEditorElement = function() {
   //DOM 요소를 작성한 값을 리턴합니다. 
};
MyCellType.prototype.getEditorValue = function(editorContext) {
   //에디터의 값을 가져옵니다. 
};
MyCellType.prototype.setEditorValue = function(editorContext, value) {
   //에디터의 값을 설정합니다. 
};
MyCellType.prototype.activateEditor = function(editorContext, cellStyle, cellRect) {
   //편집기의 특성 또는 속성 설정 및 편집기의 바인딩 이벤트 설정을 포함하여 편집기를 활성화합니다. 
};
//... 다른 메소드 구현
sheet.getRange(-1, 1, -1, 1).cellType(new MyCellType());
}
