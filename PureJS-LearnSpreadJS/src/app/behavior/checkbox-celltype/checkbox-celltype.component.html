체크박스 셀 타입 | Check Box Cell Type

1. CheckBox는 CheckBox 셀을 나타냅니다.

2. CheckBox 셀을 만들려면 다음 예제대로 하시면 됩니다. 

var c = new GC.Spread.Sheets.CellTypes.CheckBox();
sheet.setCellType(3, 2, c, GC.Spread.Sheets.SheetArea.viewport);

3. CheckBox는 isThreeState CheckBox을 지원합니다. isThreeState 메서드를 사용하여 CheckBox이 세 가지 상태를 지원하는지 여부를 가져 오거나 설정할 수 있습니다. 예 : 

c.isThreeState(true);
var state = c.isThreeState();

세 가지 상태는 true, false 또는 indeterminate입니다. 모든 상태에는 자체 텍스트가 있습니다. textTrue, textFalse 및 textIndeterminate 메서드를 사용하여 이러한 상태의 텍스트를 가져오고 설정할 수 있습니다. 예 : 

c.isThreeState(true);
c.textTrue('Check state');
c.textFalse('UnCheck state');
c.textIndeterminate('Indeterminate state');

4. caption 메서드를 사용하여 CheckBox 셀의 caption을 가져오고 설정할 수 있습니다. textAlign 메서드를 사용하여 CheckBox와 관련된 텍스트 정렬을 가져오고 설정합니다. 설정은 CheckBoxTextAlign enum 값입니다.

  • top : 텍스트가 CheckBox의 맨 위에 있습니다.
  • bottom : 텍스트가 CheckBox 아래에 있습니다.
  • left : 텍스트는 CheckBox의 왼쪽에 있습니다.
  • right : 텍스트는 CheckBox의 오른쪽에 있습니다. 
c.caption('Check Box');
c.textAlign(GC.Spread.Sheets.CellTypes.CheckBoxTextAlign.left);
