클라이언트에서 엑셀 가져오기 내보내기 | Client Side Excel IO

클라이언트 측 엑셀 가져 오기 및 내보내기는 엑셀 파일 (.xlsx)을 스프레드 json 및 스프레드 json으로 내 보냅니다.

1. ExcelIO 기능을 사용하려면 관련 js 파일 링크를 스프레드 링크가 있는 헤더 부분에 아래와 같이 추가해야합니다. 예 : 

 <head>
   ...
   <script src='.../spreadjs/gc.spread.sheets.all.x.xx.xxxxx.x.min.js' type='text/javascript'></script>>
   <script src='.../spreadjs/pluggable/gc.spread.excelio.x.xx.xxxxx.x.min.js' type='text/javascript'></script>
  </head>

2. Workbook과 IO 인스턴스를 초기화 합니다. 

var spread = new GC.Spread.Sheets.Workbook(document.getElementById('ss'));
var excelIo = new GC.Spread.Excel.IO();

3. 그러면 클라이언트 측 ExcelIO를 사용하여 excel 파일 (.xlsx)을 json으로 가져오거나 Spread json을 Excel 파일로 내보낼 수 있습니다. ExcelIO 구성 요소의 open 메서드를 사용하여 가져 오거나 ExcelIO 구성 요소의 save 메서드를 사용하여 내보낼 수 있습니다. 

//excel file을 Spread.Sheet json파일로 가져옵니다.
excelIo.open(excelFile, function (json) {
    var workbookObj = json;
    workbook.fromJSON(workbookObj);
}, function (e) {
    // process error
    console.log(e);
});
//파일을 Excel로 내보내기 위해 Spread.Sheets json을 내보냅니다.
excelio.save(json, function (blob) {
    //원하는 대로 구현하세요. 
    //당신은 저장할 수 있습니다 같이
}, function (e) {
    //process error
    console.log(e);
});

4. 또한 암호를 사용하여 Excel 파일을 열거나 저장할 수 있습니다.열기 및 저장 방법에 {password : xxxx} 옵션을 전달하면됩니다. 예 : 

//import excel file to Spread.Sheets json
excelIo.open(excelFile, function (json) {
    var workbookObj = json;
    workbook.fromJSON(workbookObj);
}, function (e) {
    // process error
    console.log(e);
},{password:xxxx});
//export Spread.Sheets json to excel file
excelio.save(json, function (blob) {
    //do whatever you want with blob
    //such as you can save it
}, function (e) {
    //process error
    console.log(e);
},{password:xxxx});
