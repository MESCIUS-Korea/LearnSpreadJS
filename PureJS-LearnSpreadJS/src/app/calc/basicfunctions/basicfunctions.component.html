기본 함수 | Basic Function

1. 스프레드 시트에서 일부 셀의 내용을 사용하는 수식을 만들어 다른 셀에 결과를 표시 할 수 있습니다. 다양한 함수를 사용하여 복잡한 수식을 쉽게 계산할 수도 있습니다.

2. 스프레드 시트는 다양한 기능을 제공합니다. : SUM과 같이 일반적으로 사용되는 몇 가지 기본 기능과 MAX와 같은 일부 고급 기능은 다음 공식 코드에서 설명하고 설명합니다. 

 //Returns the sum of the values contained in the range A1:C5.
=SUM(A1:C5)
//Returns the largest value in the range A1:C5.
=MAX(A1:C5)

= SUM (A1 : C5)을 셀에 직접 입력 할 수 있습니다. 또는 setFormula 메소드를 사용하여 수식을 적용 할 수 있습니다. 

var spread = $('#ss').data('workbook');
var sheet = spread.getActiveSheet();
// set some values to C4:C7
sheet.setFormula(8,2, '=MAX(C4:C7)');
