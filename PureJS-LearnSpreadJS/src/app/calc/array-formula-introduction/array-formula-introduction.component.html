배열함수 소개 | Array Formula Introduction

수식에는 배열에서 작동하는 함수가 포함될 수 있습니다. 배열 수식은 배열의 하나 이상의 항목에 대해 여러 계산을 수행 할 수있는 수식입니다.

Enter 키를 누르는 대신 Ctrl + Shift + Enter를 눌러 수식을 완료하므로 배열 수식을 CSE (Ctrl + Shift + Enter) 수식이라고도합니다.

Spread.Sheets는 수식의 배열 상수를 지원합니다. 중괄호 ({})를 사용하여 배열 요소를 묶으세요. 쉼표를 사용하여 행 내의 요소를 구분하고 세미콜론을 사용하여 배열 내의 행을 구분합니다. 개별 요소는 숫자 값, 텍스트 값, 논리 값 또는 오류 값일 수 있습니다. 예 : 

={1,2,3} * {4;5;6}
/* 다음과 같이 배열을 반환합니다.:
  [[4,  8, 12],
  [5, 10, 15],
  [6, 12, 18]]
*/

setArrayFormula 메서드를 사용하여 setFormula 메서드와 비슷한 배열 수식을 설정할 수 있습니다. 수식 텍스트 상자에는 중괄호 ({})로 묶인 배열 수식이 표시됩니다. 

// 배열 수식 사용 범위 설정
sheet.setArrayFormula(6,4,4,3, '=A1:A4*A1:C1');
