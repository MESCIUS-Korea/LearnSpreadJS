배열함수 사용 | Array Formula Usage

배열 수식을 사용하여 워크 시트를보다 더 강력하게 단순화하십시오. 다음 예는 배열 수식을 사용하는 방법을 보여줍니다.

1. 배열 수식은 셀의 행과 열에 대한 계산을 수행 할 수 있습니다. 그렇지 않으면 여러 수식을 사용해야 할 수도 있습니다. 예 : 

// 모든 신체의 총 점수 계산
sheet.setArrayFormula(1,5,6,1,
      '=SUBTOTAL(9,OFFSET($B$2,ROW($A$1:$A$6)-ROW($A$1),,1,3))')

2. 배열 수식을 사용하여 범위 항목을 검색 할 수 있습니다. 예 : 

// A1 : A9에 데이터가 포함되지 않은 B1 : B5의 첫 번째 값을 찾습니다.
sheet.setArrayFormula(=INDEX(B1:B5,MATCH(TRUE,ISNA(MATCH(B1:B5,A1:A9,0)),0))')

3. 배열 수식은 관련 범위 항목에서 통계 정보를 수집하는 데 도움이 될 수 있습니다. 예 : 

// Summing Sales: Brown으로 부터 판매되는 Faxes 
sheet.setArrayFormula(2,5,1,1,
     '=SUM((A2:A10="Fax")*(B2:B10="Brown")*(C2:C10))');
// 논리적 AND (Fax 및 Brown) 
sheet.setArrayFormula(5,5,1,1,
     '=SUM((A2:A10="Fax")*(B2:B10="Brown"))');
