사용자 정의 함수 | Custom Functions

1. Spread.Sheets에서 자신 만의 함수를 만들어 시트에 추가 할 수 있습니다.

2. 다음 코드와 같이 GC.Spread.CalcEngine.Functions.Function을 상속하여 고유한 함수를 만들 수 있습니다. 

function FactorialFunction() {
    this.name = 'FACTORIAL';
    this.maxArgs = 1;
    this.minArgs = 1;
 }   
FactorialFunction.prototype = new GC.Spread.CalcEngine.Functions.Function();
FactorialFunction.prototype.evaluate = function (args) }
    //테스트 할 내용
var factorial = new FactorialFunction();
sheet.addCustomFunction(factorial);

3. 사용자 고유의 함수를 추가 한 후 해당 함수를 제거하거나 모든 사용자 정의 함수를 지우려면 removeCustomFunction 또는 clearCustomFunctions 메소드를 사용하십시오 

sheet.removeCustomFunction('FACTORIAL');
sheet.clearCustomFunctions();
