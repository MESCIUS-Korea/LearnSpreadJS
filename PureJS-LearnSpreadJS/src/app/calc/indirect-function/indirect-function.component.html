Indirect 함수 | Indirect Function

Spread.Sheets는 Microsoft Excel과 유사한 INDIRECT 기능을 제공합니다.

INDIRECT는 텍스트 문자열로 지정된 레퍼런스를 반환합니다. 레퍼런스가 즉시 평가되어 내용을 표시합니다. 수식 자체를 변경하지 않고 수식 내의 셀에 대한 레퍼런스를 변경하려면 INDIRECT를 사용하십시오.

INDIRECT는 a1 스타일 레퍼런스, r1c1 스타일 레퍼런스, 명명 된 레퍼런스 또는 셀에 대한 레퍼런스를 텍스트 문자열로 지원합니다. 셀 레퍼런스가 잘못된 경우 INDIRECT는 #REF! 오류 값을 반환합니다.
