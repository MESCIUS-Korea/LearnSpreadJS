와일드 카드 문자 | Function Wildcards

검색시 함수의 비교 기준으로 와일드 카드 문자 (?, *, ~)를 사용할 수 있습니다.

       Wildcard character                            Match                     
? 모든 단일 문자
* 임의의 문자 수
~ 뒤이어 ?, *, or ~ ?, *, or ~ character


참고:와일드 카드는 equals (=)를 사용하는 비교 문자열에서만 사용할 수 있습니다. 다른 연산자 (>, <,! =, <=,> =)를 사용하는 비교에서는 사용할 수 없습니다..

와일드 카드 문자는 다음과 같이 기준 인수가 있는 모든 함수에서 작동합니다.

  • AVERAGEIF
  • AVERAGEIFS
  • COUNTIF
  • COUNTIFS
  • SUMIF
  • SUMIFS
  • MATCH
  • SEARCH
  • VLOOKUP
  • HLOOKUP
  • DAVERAGE
  • DGET
  • DMAX
  • DMIN
  • DPRODUCT
  • DSTDEV
  • DSTDEVP
  • DSUM
  • DVAR
  • DVARP
