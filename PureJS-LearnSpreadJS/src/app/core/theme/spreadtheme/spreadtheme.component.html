Spread 테마 | Sheet Theme

1.Spread.Sheets는 다음과 같이 CSS 파일에 의해 제공되는 Excel의 다양한 스타일을 지원합니다. :

  • Excel 2007 : gc.spread.sheets.x.xx.xxxxx.x.css
  • Excel 2013 White : gc.spread.sheets.excel2013white.x.xx.xxxxx.x.css
  • Excel 2013 Light Gray : gc.spread.sheets.excel2013lightgray.x.xx.xxxxx.x.css
  • Excel 2013 Dark Gray : gc.spread.sheets.excel2013darkgray.x.xx.xxxxx.x.css
  • Excel 2016 Colorful : gc.spread.sheets.excel2016colorful.x.xx.xxxxx.x.css
  • Excel 2016 Dark Gray : gc.spread.sheets.excel2016darkgray.x.xx.xxxxx.x.css

2. 페이지의 Head 부분에 스타일 링크를 추가하여 그 중 하나를 사용할 수 있습니다. 예: 

<link rel='stylesheet' href="css/gc.spread.sheets.excel2013white.x.xx.xxxxx.x.css"/>


3. CSS 참조 파일을 변경하여 테마간에 동적 전환이 가능합니다. 변경 사항을 적용하려면 Spread의 통화 새로 고침 방법이 필요합니다.


샘플코드:

결과화면과 함께 HTML, JS 코드도 확인이 가능합니다.
