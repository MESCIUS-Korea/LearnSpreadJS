확장 테마 | External Theme

1. 기본 Spread.Sheets 테마와 함께 외부 테마를 사용할 수 있습니다.

2. Spread.Sheets에서 부트 스트랩 테마를 사용할 수 있습니다. 이를 위해서는 스타일 링크가 Spread.Sheets 스타일 링크 다음에 페이지의 헤드 섹션에 추가되어야 합니다. 예를 들어, css 폴더의 프로젝트에서 bootstrap.css 및 bootstrap-theme.css가있는 경우 다음 스타일 링크를 추가 할 수 있습니다. 

<link rel="stylesheet" href="css/gc.spread.sheets.x.xx.xxxxx.x.css"/>

<link rel="stylesheet" href="css/bootstrap.css"/>
<link rel="stylesheet" href="css/bootstrap-theme.css"/>

3. Spread.Sheets는 jQuery UI 테마를 지원합니다. 이를 위해서는 jQuery UI 스타일 링크가 Spread.Sheets 스타일 링크 다음에 페이지의 헤드 섹션에 추가되어야 합니다. 예를 들어 jQuery UI sunny 테마를 사용하도록 다음 스타일 링크를 추가 할 수 있습니다. 

<link rel="stylesheet" href="css/gc.spread.sheets.x.xx.xxxxx.x.css"/>

<link href="css/sunny/jquery-ui.css" rel="stylesheet"  type="text/css"/>

샘플코드:

결과화면과 함께 HTML, JS 코드도 확인이 가능합니다.
