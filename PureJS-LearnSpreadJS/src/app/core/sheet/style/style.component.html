스타일 | Style

Spread.Sheets는 foreColor, backColor 등과 같은 많은 속성을 포함하는 Style 클래스를 제공합니다.

1. 다음과 같은 코드를 사용하여 스타일을 만들고 속성을 설정할 수 있습니다. : 

var style = GC.Spread.Sheets.Style();
style.backColor = 'red';
style.foreColor = 'green';

2. 그런 다음 스타일을 셀, 행 또는 열로 설정할 수 있습니다 : 

//셀에 스타일적용.
sheet.setStyle(5, 5, style, GC.Spread.Sheets.SheetArea.viewport);
//행에 스타일 적용.
sheet.setStyle(5, -1, style, GC.Spread.Sheets.SheetArea.viewport);
//열에 스타일 적용.
sheet.setStyle(-1, 5, style, GC.Spread.Sheets.SheetArea.viewport);

각 레벨의 스타일은 다음과 같이 우선 순위가 다릅니다. cell> row> column.

3. Spread.Sheets를 사용하면 스타일의 이름을 설정하고 이 명명 된 스타일을 시트의 명명 된 스타일 컬렉션에 추가 할 수 있습니다. 따라서 스타일을보다 편리하게 사용하고 관리 할 수 있습니다.

스타일의 이름을 설정하고 시트 또는 Spread 구성 요소의 명명 된 스타일 모음에 이 스타일을 추가 할 수 있습니다. 

var style = new GC.Spread.Sheets.Style();
style.name = 'style1';
style.backColor = 'red';

//시트의 명명 된 스타일 컬렉션에 추가
sheet.addNamedStyle(style);
//스프레드의 명명 된 스타일 컬렉션에 추가
spread.addNamedStyle(style)

명명 된 스타일이 명명 된 스타일 모음에 추가되면 이름으로 스타일을 가져올 수 있습니다 : 

sheet.getNamedStyle('style1');
spread.getNamedStyle('style1');

명명 된 스타일을 사용하지 않으면 명명 된 스타일 컬렉션에서 제거 할 수 있습니다 :

sheet.removeNamedStyle('style1');
spread.removeNamedStyle('style1');
