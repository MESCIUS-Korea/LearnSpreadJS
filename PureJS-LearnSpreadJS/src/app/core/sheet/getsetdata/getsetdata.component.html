Get/Set 데이터 | Get Set Data

Spread.Sheets은 시트를 통해 셀의 데이터로 작업 할 수 있는 다양한 방법을 제공합니다.

1. 지정한 셀의 값을 가져 오거나 설정할 수 있습니다. 

sheet.setValue(7,7,'2014');
sheet.getValue(7,7);

일련의 값을 가져 오거나 셀 범위에 값을 설정하려면 다음과 같은 코드를 사용할 수 있습니다: 

sheet.setArray(7, 7, [['value1','value2','value3','value4''], ['value5','value6','value7','value8']]);
sheet.getArray(7, 7, 2, 4);

2. 지정된 셀에 텍스트를 가져 오거나 설정할 수도 있습니다. 이는 셀에 값을 가져 오거나 설정하는 것과 유사합니다. 

sheet.setText(7, 7, '01/01/2014');
sheet.getText(7, 7);

3. 일부 데이터를 삭제하려면 다음 방법을 사용하여 지정된 범위의 데이터를 지울 수 있습니다. 

sheet.clear(0, 0, 10, 10, GC.Spread.Sheets.SheetArea.viewport, GC.Spread.Sheets.StorageType.data);

4. 스프레드 시트는 한 범위에서 다른 범위로 일련의 데이터를 복사하거나 이동하는 기능도 지원합니다. 

sheet.copyTo(0, 0, 14, 14, 7, 7, GC.Spread.Sheets.CopyToOptions.all);
sheet.moveTo(0, 0, 14, 14, 7, 7, GC.Spread.Sheets.CopyToOptions.all);

결과화면과 함께 HTML, JS 코드도 확인이 가능합니다.
