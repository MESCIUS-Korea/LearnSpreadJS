보호 | Protection

1. 때때로 이 시트의 셀 값을 편집하지 않기를 원합니다. 그렇다면 isProtected를 사용하여 시트가 보호되어 있는지 여부를 설정하고 가져올 수 있습니다.

var spread = $('#ss').data('workbook');
var sheet = spread.getActiveSheet();
sheet.options.isProtected =  true;
sheet.options.isProtected;   //true

2. 시트가 보호되면 protectionOption을 사용하여 다음과 같은 속성을 가진 객체를 제공하여 사용자 동작을 제한 할 수 있습니다. :

allowSelectLockedCells : 사용자가 잠긴 셀을 선택할 수 있는지 여부를 나타내는 Boolean입니다.

allowSelectUnlockedCells : 사용자가 잠금 해제 된 셀을 선택할 수 있는지 여부를 나타내는 Boolean입니다.

allowSort : 사용자가 범위를 정렬 할 수 있는지 여부를 나타내는 Boolean입니다.

allowFilter : 사용자가 범위를 필터링 할 수 있는지 여부를 나타내는 Boolean입니다.

allowEditObjects : 사용자가 부동 개체를 편집 할 수 있는지 여부를 나타내는 Boolean입니다.

allowResizeRows : 사용자가 행의 크기를 조정할 수 있는지 여부를 나타내는 Boolean입니다.

allowResizeColumns : 사용자가 열의 크기를 조정할 수 있는지 여부를 나타내는 Boolean입니다. 

var spread = $('#ss').data('workbook');
var sheet = spread.getActiveSheet();
sheet.options.protectionOption = { 
    allowSelectLockedCells: true,
    allowSelectUnlockedCells: true,
    allowSort: false 
};
var option = sheet.options.protectionOption;
console.log(JSON.stringify(option, null, 2));

결과화면과 함께 HTML, JS 코드도 확인이 가능합니다.
