Sheet 헤더 | Sheet Header

스프레드 시트에서 시트는 4 개의 영역으로 구성됩니다. :

  • corner
  • colHeader
  • rowHeader
  • viewport

따라서 시트의 머리글에는 행 머리글과 열 머리글이 포함되며 행, 열 머리글에는 여러 행 / 열이 포함될 수 있습니다.

1. 머리글의 행이나 열 중 하나에서 각 셀에는 열 또는 행의 인덱스를 나타내는 자동 텍스트 ( "자동 텍스트")가 있습니다. 어떤 행이나 열에 자동 텍스트가 표시되는지 알고 싶거나 특정 행이나 열에 자동 텍스트를 표시하려면 다음과 같은 코드를 사용하십시오 : 

sheet.options.rowHeaderAutoTextIndex = 0; 
sheet.options.rowHeaderAutoTextIndex;

sheet.options.colHeaderAutoTextIndex = 0;
sheet.options.colHeaderAutoTextIndex;

2. 스프레드는 빈칸, 글자 및 숫자 중에서 선택할 수 있는 세 가지 유형의 헤더 자동 텍스트도 제공합니다. 

sheet.options.rowHeaderAutoText = GC.Spread.Sheets.HeaderAutoText.letters;
sheet.options.colHeaderAutoText = GC.Spread.Sheets.HeaderAutoText.blank;

행 머리글이나 열 머리글을 숨기도록 선택할 수 있습니다. 다음 코드는 행 헤더의 모든 행을 숨 깁니다. 

sheet.options.rowHeaderVisible = false;

샘플코드:

결과화면과 함께 HTML, JS 코드도 확인이 가능합니다.
