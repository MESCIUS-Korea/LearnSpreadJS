셀 병합 | Cell Span

병합은 둘 이상의 셀을 결합하여 더 큰 새로운 셀을 만듭니다. 여러 열에 걸쳐 라벨을 만드는 훌륭한 방법입니다.

1. 다음 코드와 같이 addSpan 메서드를 사용하여 지정된 범위의 셀을 새롭고 큰 셀에 병합합니다. 

// 셀 병합 및 레이블 설정
sheet.addSpan(1, 4, 1, 7);
sheet.setValue(1, 4, "Goods");
// 행 (3) 및 열 (4)에 셀 병합
sheet.addSpan(20, 1, 3, 4);
sheet.getCell(20, 1).value("Demo").hAlign(1).vAlign(1);

2. removeSpan 메서드를 사용하여 지정된 셀이 포함 된 범위를 제거합니다. 

sheet.removeSpan(20, 1);

참고 : 병합 된 셀에 표시 할 데이터가 왼쪽 위 셀에 있는지 확인하십시오. 병합 된 다른 셀의 모든 데이터는 Span이 제거 될 때까지 숨겨집니다.


샘플코드:

결과화면과 함께 HTML, JS 코드도 확인이 가능합니다.
