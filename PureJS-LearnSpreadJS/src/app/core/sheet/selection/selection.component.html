셀 선택 | Cell Selection

1. 하나의 셀을 클릭하고 마우스를 끌어 범위를 선택하십시오. 그런 다음 범위 선택을 할 수 있게 됩니다.

다음 코드와 같이 selectionBorderColor 및 selectionBackColor 옵션을 사용하여 선택 영역 테두리 색과 배경색을 변경할 수 있습니다. : 

var spread = $('#ss').data('workbook'); 
var sheet = spread.getActiveSheet();
sheet.options.selectionBorderColor ='red';
sheet.options.selectionBackColor = transparent';

3. 사용자가 selectionPolicy 및 selectionUnit 옵션을 사용하여 선택할 수 있는 항목을 설정할 수 있습니다. 열거 형 SelectionPolicy는 세 가지 유형:

  • single: 단일 항목 만 선택할 수 있습니다.
  • range: 단일 항목 및 항목 범위를 선택할 수 있지만 여러 범위는 선택할 수 없습니다.
  • muliRange: 여러 항목을 포함하여 단일 항목 및 항목 범위를 선택할 수 있습니다.

열거 형 SelectionUnit에는 세 가지 유형이 있습니다.

  • 셀 : 선택할 수 있는 가장 작은 단위가 셀임을 나타냅니다.
  • 행 : 선택할 수 있는 가장 작은 단위가 행임을 나타냅니다.
  • 열 : 선택할 수 있는 가장 작은 단위가 열임을 나타냅니다.

이 두 가지 방법을 사용하여 선택 모드를 제어 할 수 있습니다.

4. Ctrl 키를 누르고 범위를 선택하십시오. 여러 범위를 선택하게 됩니다. 또한 addSelection 메서드를 사용하여 선택 항목을 더 추가 한 다음 getSelections 메서드를 사용하여 선택한 모든 범위를 가져올 수 있습니다. 선택을 지우려면 clearSelection 메소드를 사용하십시오. 이러한 메소드는 다음 코드에서 사용됩니다. : 

sheet.addSelection(0, 0, 3, 3);
sheet.addSelection(4, 3, 2, 2);
sheet.clearSelection();
var selections = sheet.getSelections(); 
// clearSelection의 뒤, selections.length는 0입니다.

5. 마우스를 사용하여 선택하는 것 외에도 setSelection 메서드를 사용하여 일부 셀을 선택하고 setActiveCell 메서드를 사용하여 셀 하나를 선택할 수 있습니다. 활성 셀은 선택 영역의 첫 번째 셀입니다. 다음 코드와 같이 getActiveRowIndex 및 getActiveColumnIndex 메서드를 사용하여 활성 셀 행 및 열 인덱스를 가져옵니다. : 

sheet.addSelection(0, 0, 3, 3);
sheet.setActiveCell(20, 3, 2, 2);
var activeRowIndex = sheet.getActiveRowIndex();
var activeColumnIndex = sheet.getActiveColumnIndex();

활성 셀을 설정 한 후 활성 셀이 표시되지 않으면 showCell, showRow 및 showColumn 메서드를 사용하여 활성 셀을 표시되도록 할 수 있습니다. 

var verticalPosition = GC.Spread.Sheets.VerticalPosition.top;
var horizontalPosition = GC.Spread.Sheets.HorizontalPosition.left;
sheet.showCell(20, 3, verticalPosition, horizontalPosition);
// up 코드 및 down 코드가 같은 결과를 낸다..
sheet.showColumn(3, horizontalPosition);
sheet.showRow(20, verticalPosition);
