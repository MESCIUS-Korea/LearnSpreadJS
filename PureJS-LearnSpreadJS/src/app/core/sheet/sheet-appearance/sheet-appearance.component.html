Sheet 외관 | Sheet Appearance

Spread는 보이는, 선 테두리 및 확대 / 축소를 포함하여 시트 모양을 변경하는 몇 가지 인터페이스를 제공합니다.

1. Spread.Sheets에서 시트를 숨기려면 다음 코드 만 있으면 됩니다. : 

sheet.visible(false);

2. Spread는 여러 유형의 선 스타일을 제공하여 얇은 점선, 이중선 등과 같이 시트의 경계선과 그리드 선을 사용자 정의하는 데 도움을 줍니다. 예를 들어, 다음 행은 표시 범위의 왼쪽 및 오른쪽 경계선을 점선 스타일로 변경합니다. 

var lineStyle = GC.Spread.Sheets.LineStyle.dotted; 
var lineBorder = new GC.Spread.Sheets.LineBorder('red', lineStyle);
var sheetArea = GC.Spread.Sheets.SheetArea.viewport
sheet.getRange(1, 1, 1, 1).setBorder(lineBorder, { left: true, right: true }, sheetArea);

3. 확대 / 축소 방법을 사용하여 현재 시트의 배율을 변경합니다. 

sheet.zoom(3);

샘플코드:

결과화면과 함께 HTML, JS 코드도 확인이 가능합니다.
