Sheet 페인트 | Sheet Paint

1. 사용자가 시트를 변경 한 후에는 Spread 구성 요소가 이러한 변경 사항을 처리하고 새 시트를 칠해야 합니다.

2. repaint 메서드를 사용하여 시트를 다시 칠할 수도 있습니다. 

var spread = $('#ss').data('workbook');
var sheet = spread.getActiveSheet();
sheet.repaint();

대부분 시트를 변경하면 구성 요소가 자동으로 새로 고침 되어 변경 내용에 반응합니다. 한 번에 많이 변경했지만 시트를 여러 번 다시 칠하지 않으려면 suspendPaint 메서드를 사용하여 변경이 완료 될 때까지 자동 다시 칠을 중지 할 수 있습니다. 다음 예제와 같이 resumePaint 메서드를 호출하여 시트를 다시 그립니다.: 

var spread = $('#ss').data('workbook');
var sheet = spread.getActiveSheet();
sheet.suspendPaint();
sheet.setValue(2, 2, '나를 클릭하고 키보드로 문자를 입력하세요'); 
sheet.addSpan(2, 2, 1, 5); 
sheet.setValue(4, 2, '검은 색 빈 셀을 두 번 클릭하십시오!'); 
sheet.addSpan(4, 2, 1, 5); 
//...
sheet.resumePaint();
