Sheet 초기화 | Initialize Sheet

스프레드 시트에서 각 시트 영역에는 여러 행과 열이 있습니다. Spread.Sheets은 행과 열을 사용자 정의 할 수 있는 메소드를 제공합니다.

1. setRowCount 및 setColumnCount 메서드를 사용하여 각 시트 영역의 행 또는 열 수를 변경합니다. 또한 addRows, addColumns, deleteRows 및 deleteColumns 메서드를 사용하여 뷰포트의 행 또는 열 수를 변경할 수 있습니다.

다음의 각 메소드는 뷰포트 영역의 행 수를 변경하는 데 사용될 수 있습니다. 

sheet.setRowCount(20, GC.Spread.Sheets.SheetArea.viewport);
sheet.addRows(5, 10);
sheet.deleteRows(20, 10);

2. 행 또는 열 크기 조정 가능 속성이 false 인 경우 사용자 조작으로 크기를 조정할 수 없지만 코드를 사용하여 높이 또는 너비를 직접 변경할 수 있습니다. 

sheet.setRowResizable(1, false);
sheet.setColumnResizable(1, true);

sheet.setRowHeight(1, 20);
sheet.setColumnWidth(1, 40);

3. 행 또는 열을 표시할지 여부를 지정하려면 setRowVisible 및 setColumnVisible 메서드를 사용하십시오. 

sheet.setRowVisible(1, false);
sheet.setColumnVisible(1, true);

4. 시트의 뷰포트 영역에서 행 또는 열이 자동으로 맞을 때 높이 또는 너비는 내용의 길이에 따라 결정됩니다. 행 또는 열을 자동 맞춤으로 설정하려면 다음 코드를 사용하십시오. : 

sheet.autoFitColumn(1);
sheet.autoFitRow(1)

5. Spread.Sheets는 또한 시트의 행이나 열에 대한 유용한 정보를 얻기 위한 여러 가지 방법을 제공합니다 (코드에 설명되어 있음). 

var rowCount = sheet.getRowCount();
var columnCount = sheet.getColumnCount();
var cell = sheet.getCell(7, 7);
var cells = sheet.getCells(2, 2, 7, 7);
var rowHeight = sheet.getRowHeight(7);
var columnWidth = sheet.getColumnWidth(7);
...

6. Spread.Sheets는 행 높이 또는 열 너비가 0 일 때 행 또는 열 머리글에 표시 될 double 또는 단일 눈금 선을 제어하는 resizeZeroIndicator 메서드를 제공합니다. 매개 변수는 ResizeZeroIndicator 열거 형 값입니다.

  • default: 단일 눈금 선 (표준과 동일)
  • enhanced: 이중 눈금 선 (기본값)
sheet.setRowHeight(4, 0);
sheet.setColumnWidth(2, 0);
spread.resizeZeroIndicator(GC.Spread.Sheets.ResizeZeroIndicator.enhanced);
//spread.resizeZeroIndicator(GC.Spread.Sheets.ResizeZeroIndicator.default);
