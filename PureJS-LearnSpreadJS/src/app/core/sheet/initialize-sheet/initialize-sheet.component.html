Sheet 초기화 | Initialize Sheet

1. 다른 라이브러리와 마찬가지로 Sheet에는 다음과 같은 포함 파일이 필요합니다.

  • gc.spread.sheets.x.xx.xxxxx.x.css
  • gc.spread.sheets.all.x.xx.xxxxx.x.min.js

2. 다음 코드를 사용하여 시트 만들기 (매개 변수는 시트의 이름입니다). 

var sheet = new GC.Spread.Sheets.Sheet('NewSheet');

3. 기존 Spread 구성 요소의 지정된 위치에 추가합니다. 

spread.addSheet(0, sheet);

4. Spread 구성 요소에서 기존 시트를 제거하려면 다음 코드를 사용하십시오. 이 예제 코드는 Spread의 첫 번째 시트를 제거합니다. 

spread.removeSheet(0, sheet);

5. Spread 구성 요소의 모든 시트를 제거하려면 편리한 clearSheets 메소드를 사용하십시오. 

spread.clearSheets();

6. 시트의 이름을 사용자 정의하려면 이름 메소드를 사용하여 시트의 이름을 가져오고 설정하십시오. 

var  sheetName = sheet.name();
sheet.name('sheet1');
