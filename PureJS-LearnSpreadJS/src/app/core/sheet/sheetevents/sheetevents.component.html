Sheet 이벤트 | Sheet Events

1. Spread.Sheets는 시트에 대한 이벤트를 제공합니다. 이벤트가 발생할 때 응용 프로그램에서 작업을 수행하게 할 수 있습니다.

2. 시트의 바인드 및 바인드 해제 메소드를 사용하여 시트 이벤트를 바인드 할 수 있습니다. unbindAll 메소드를 사용하여 모든 이벤트를 바인드 해제 할 수도 있습니다. jquery의 바인드 및 바인드 해제와 마찬가지로 다음 코드와 같이 시트의 바인드 및 바인드 해제를 처리 할 수 있습니다 : 

var spread = $('#ss').data('workbook');
var sheet = spread.getActiveSheet();
var selectionChanged = GC.Spread.Sheets.Events.SelectionChanged;
sheet.bind(selectionChanged, function(e,args) { // sheet event
// do some thing...
});
// 시트 이벤트 바인딩을 제거
sheet.unbind(selectionChanged); 
시트의 모든 이벤트 바인딩을 제거
sheet.unbindAll();

3. 여러 이벤트를 트리거 할 수 있는 태스크를 수행하려는 경우 시트가 이러한 이벤트를 트리거하지 않게 하려면 suspendEvent 메소드를 사용하여 이벤트가 발생하지 않도록 하십시오. 작업이 완료되면 resumeEvent 메서드를 사용하여 다음 예제와 같이 트리거 이벤트를 다시 시작할 수 있습니다. : 

var spread = $('#ss').data('workbook');
var sheet = spread.getActiveSheet();
sheet.suspendEvent();
//do some operation...
sheet.resumeEvent();
