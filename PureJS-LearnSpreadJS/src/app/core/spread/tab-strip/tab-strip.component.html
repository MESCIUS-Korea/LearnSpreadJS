탭 스트립 | TabStript

1. 스프레드시트 탭 스트립은 Excel과 유사합니다. 탐색 버튼을 클릭하여 시트 탭을 이동하고 시트 탭을 클릭하여 활성 시트를 변경하고 시트 탭을 드래그하여 시트 목록의 순서를 변경 한 다음 탭 스트립 스플리터를 드래그하여 탭 스트립 폭의 크기를 조정할 수 있습니다. 또한 탭 스트립을 사용자 정의 할 수 있습니다.


2. tabStripVisible을 설정하여 탭 스트립을 표시할지 여부를 제어 할 수 있습니다. 또한 새 탭을 표시하지 않으려면 newTabVisible을 false로 설정하십시오.

var spread = $('#ss').data('workbook');
spread.options.tabStripVisible = false;
//Also you can set tabStripVisible to true.   
spread.options.newTabVisible = false;
//Also you can set newTabVisible to true.

3. 시트의 이름 목록은 탐색 버튼 바로 뒤에 표시됩니다. 시트 탭을 두 번 클릭하여 시트의 이름을 바꿀 수 있습니다. 이름을 두 번 클릭하거나 편집하여 시트 이름을 변경하지 않으려면 펼치기 옵션 tabEditable을 false로 설정하십시오. 스프레드 옵션 allowSheetReorder를 설정하여 시트 탭을 드래그하여 시트 목록의 순서를 변경할 수 있는지 여부를 제어 할 수 있습니다. 

spread.options.tabEditable = true; 
spread.options.allowSheetReorder = true;

스프레드 옵션 인 sheetTabColor를 사용하여 시트 탭의 색상을 사용자 정의 할 수 있습니다. 

var sheet = spread.getActiveSheet();
sheet.options.sheetTabColor = 'red';

4. 탭 스트립 스플리터를 드래그하여 탭 스트립의 너비를 조정할 수 있습니다. 또한 펼침 옵션 tabStripRatio를 사용하여 탭 스트립의 너비 비율을 가져오고 설정할 수 있습니다.

var ratio = spread.options.tabStripRatio;
 spread.options.tabStripRatio = 0.6;

5. 펼치기 옵션 tabNavigationVisible을 설정하여 show (default)를 제어하거나 탭 네비게이션 관련 항목을 숨길 수 있습니다. 

spread.options.tabNavigationVisible = false; // false: hide, true: show
