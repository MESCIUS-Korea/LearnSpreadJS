데이터 소스 Serialize | Data Source Serialize

1. includeBindingSource 속성이 true로 설정된 객체를 사용하여 Spread.toJSON을 호출하면 시트 또는 테이블의 데이터 소스를 JSON에 직렬화 할 수 있습니다. 

sheet.setDataSource(dataSource);
var spread = $('#ss').data('workbook');
var jsonString = JSON.stringify(
      spread1.toJSON( { includeBindingSource:true } )
);
