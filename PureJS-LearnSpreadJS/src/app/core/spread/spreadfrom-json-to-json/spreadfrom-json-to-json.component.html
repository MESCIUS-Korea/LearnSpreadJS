Spread JSON 데이터 | Spread from JSON / toJSON

1. Spread.Sheets는 JSON 직렬화 및 비 직렬화를 지원합니다. 스프레드를 JSON에 저장하고 JSON을 열어 스프레드를 초기화 할 수 있습니다.

2. JSON.stringify를 사용하여 스프레드의 toJSON에서 오는 JSON 객체를 직렬화 할 수 있습니다. 예: 

var spread = $('#ss').data('workbook');
var jsonString = JSON.stringify(spread1.toJSON());

3. 그런 다음 JSON.parse를 사용하여 JSON 문자열을 비 직렬화하고 fromJSON을 사용하여 Spread 객체를 초기화합니다. 그러면 spread1과 같은 새로운 스프레드가 생깁니다. 예: 

var spread2 = $('#ss1').data('workbook');
spread2.fromJSON(JSON.parse(jsonStr));

결과화면과 함께 HTML, JS 코드도 확인이 가능합니다.
