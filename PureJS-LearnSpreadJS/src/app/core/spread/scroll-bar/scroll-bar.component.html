스크롤바 | Scroll Bar

1.Spread.Sheets의 가로 및 세로 스크롤 바를 사용하여 시트를 이동하여 시트의 일부를 볼 수 있습니다.


2. 스크롤 바는 화살표 버튼, 스크롤 박스 및 스크롤 박스 컨테이너로 구성됩니다. Spread의 스크롤 바를 사용하면 화살표 버튼을 누르거나 스크롤 박스를 드래그하거나 스크롤 박스와 화살표 버튼 사이의 간격을 클릭 할 수 있습니다. 이러한 동작은 다음과 같은 결과를 제공합니다:

  • 한 번에 한 행 또는 열을 스크롤하려면 화살표 단추를 누릅니다.
  • 스크롤 박스를 위아래로 드래그하여 시트를 위아래로 스크롤합니다 (또는 왼쪽 및 오른쪽).
  • 스크롤 박스와 화살표 단추 사이의 간격을 클릭하여 시트를 한 페이지 스크롤합니다.

3. Spread.Sheets는 스크롤 박스를 끌 때 스크롤 도구 설명을 표시 할 수 있습니다. showScrollTip 옵션 항목을 사용하여 표시 할 스크롤 도구 설명 유형을 설정할 수 있습니다. 스크롤 툴팁은 스크롤 박스를 끌 때 뷰의 맨 위 행 또는 왼쪽 열의 번호를 표시합니다. 

var spread = $('#ss').data('workbook');
spread.options.showScrollTip = GC.Spread.Sheets.ShowScrollTip.horizontal;
spread.options.showScrollTip = GC.Spread.Sheets.ShowScrollTip.vertical;
spread.options.showScrollTip = GC.Spread.Sheets.ShowScrollTip.both;
spread.options.showScrollTip = GC.Spread.Sheets.ShowScrollTip.none;

3. showVerticalScrollbar 및 showHorizontalScrollbar 옵션 항목을 사용하여 스크롤 바를 표시할지 여부를 제어 할 수 있습니다.

var spread = $('#ss').data('workbook');
spread.options.showVerticalScrollbar = false;
spread.options.showHorizontalScrollbar = false;

샘플코드:

결과화면과 함께 HTML, JS 코드도 확인이 가능합니다.
