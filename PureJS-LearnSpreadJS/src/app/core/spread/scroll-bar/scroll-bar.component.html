스크롤바 | Scroll Bar

1. 스프레드 시트는 의존성이 없습니다. 다음 파일이 필요합니다.

  • gc.spread.sheets.x.xx.xxxxx.x.css
  • gc.spread.sheets.all.x.xx.xxxxx.x.min.js

2. gc.spread.sheets.x.xx.xxxxx.x.css 와 gc.spread.sheets.all.x.xx.xxxxx.x.min.js를 파일들의 상대경로를 문서의 head 섹션에 포함시킵니다. 예:

<!DOCTYPE html>
<html>
   <head>
      <link href='gc.spread.sheets.x.xx.xxxxx.x.css' rel='stylesheet' type='text/css'/>
      <script src='gc.spread.sheets.all.x.xx.xxxxx.x.min.js' type='text/javascript'></script>
      <script type='text/javascript' charset='utf-8'>
      ...
      </script>
   </head>
   <body>
        <div id='ss' style='width:100%; height:400px;'></div>
   </body>
</html>

3. 스프레드 구성 요소는 새로운 GC.Spread.Sheets.Workbook(document.getElementById('ss'), { sheetCount: 1 }), 로 초기화되고 스프레드는 'ss' id로 div에 추가됩니다.
여러분은 다음과 같은 방법으로 스프레드 객체를 얻을 수 있습니다: var spread = GC.Spread.Sheets.findControl(document.getElementById('ss')).
이후, 스프레드를 커스터마이즈 할 수 있습니다. 예: 

window.onload = function () {
   var spread = new GC.Spread.Sheets.Workbook(document.getElementById('ss'), { sheetCount: 1 });
   // get spread object
   // var spread = GC.Spread.Sheets.findControl(document.getElementById('ss'));
};

4. jQuery와 함께 Spread.Sheets를 사용할 수 있습니다. 예:

$(document).ready(function () {
   var spread = new GC.Spread.Sheets.Workbook($('#ss')[0], { sheetCount: 1 });
   // get spread object
   // var spread = $('#ss').data('workbook');
});
