스프레드시트 | SpreadSheet

스프레드 시트는 시트, 탭 스트립, 스크롤바 등으로 구성됩니다. 핵심 부분은 시트입니다. 시트를 사용자 정의 할 수 있습니다.

1. 더 많은 시트를 추가하여 더 많은 데이터를 배치하려면 빈 시트를 추가 할 색인 매개 변수를 설정하거나 두 번째 매개 변수를 추가 할 시트 인 두 개의 매개 변수를 설정하십시오. 예 : 

var spread = $('#ss').data('workbook');
spread.addSheet(spread.getSheetCount());
var spread = new GC.Spread.Sheets.Worksheet(''New Sheet'')
spread.addSheet(0, sheet);

2. 시트 수를 사용자 정의 할 수 있습니다. getSheetCount 및 setSheetCount를 사용하여 시트 수를 가져오고 설정하십시오. 예 :

spread.setSheetCount(4);
var sheetCount = spread.getSheetCount();

3. 펼칠 시트를 추가 한 후 getSheet 또는 getSheetFromName으로 시트를 가져올 수 있습니다. 시트 이름은 시트 탭 이름과 동일합니다. 예 : 

var sheet = spread.getSheet(0);
var sheet1 = spread.getSheetFromName('Sheet1');

4. 일부 시트가 필요하지 않은 경우 removeSheet 및 clearSheets를 사용하십시오. 예 :

spread.removeSheet(0);
spread.clearSheets();

5. 활성 시트는 현재 작업하고 있는 시트입니다. getActiveSheet를 사용하여 활성 시트를 가져온 다음 설정을 사용자 정의 할 수 있습니다. getActiveSheetIndex 및 setActiveSheetIndex를 사용하여 활성 시트 색인을 가져 와서 설정할 수도 있습니다. 예 : 

var spread = $('#ss').data('workbook');
var sheet = spread.getActiveSheet();
// customize sheet
spread.setActiveSheetIndex(1); // change the active sheet.
var activeSheetIndex = spread.getActiveSheetIndex();

샘플코드:

결과화면과 함께 HTML, JS 코드도 확인이 가능합니다.
