Spread 이벤트 | Spread Events

1. 스프레드시트는 이벤트 중에 응답 할 수 있는 이벤트를 제공합니다.

2. 스프레드 시트와 시트 이벤트를 바인딩 할 수 있습니다. 바인드로 이벤트를 bind하고 unbind로 이벤트를 바인드 해제 할 수 있습니다. unbindAll을 사용하여 모든 이벤트를 바인드 해제 할 수 있습니다. jquery의 바인딩 및 바인딩 해제와 마찬가지로 스프레드의 바인드 및 바인딩 해제를 처리 할 수 있습니다. 예 

var spread = $('#ss').data('workbook');
var sheet = spread.getSheet(0);
var activeSheetChanged = GC.Spread.Sheets.Events.ActiveSheetChanged;
spread.bind(activeSheetChanged, function(e,args) {  // spread event

});
var selectionChanged = GC.Spread.Sheets.Events.SelectionChanged
sheet.bind(selectionChanged, function (e,args) {  // sheet event

});

3. 스프레드 시트와 시트에서 이벤트를 트리거하지 않으려면 suspendEvent를 사용하십시오. resumeEvent를 사용하여 트리거 이벤트를 다시 시작하십시오. 예 

var spread = $('#ss').data('workbook');
spread.suspendEvent();
 //do some operation...
spread.resumeEvent();
