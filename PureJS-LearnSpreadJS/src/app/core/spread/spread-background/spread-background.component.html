Spread 배경화면 | Spread Background

1. 스프레드 시트의 배경 모양을 사용자 정의 할 수 있습니다.


2. backColor 또는 backgroundImage를 사용하여 모든 시트의 배경색 또는 배경 이미지를 설정할 수 있습니다. 배경색과 배경 이미지를 모두 설정하면 배경 이미지가 표시됩니다. 이미지가 표시됩니다. 

var spread = $('#ss').data('workbook');
spread.options.backColor = 'red';
// The backColor will work if the backgroundImage not set.  
spread.options.backgroundImage = 'images/back.PNG';

Spread.Sheets는 backgroundImageLayout을 사용하여 이미지 레이아웃을 제어하는 기능을 제공합니다. 이미지 레이아웃에는 다음 유형이 포함됩니다. :

  • stretch: 영역의 크기가 변경 되더라도 배경 이미지가 영역을 채웁니다.
  • center: 배경 이미지가 영역의 중앙에 표시됨을 나타냅니다.
  • zoom: 원래의 비율을 유지하면서 배경 이미지가 영역에 표시됨을 나타냅니다.
  • none: 배경 이미지가 원본 영역이있는 영역의 왼쪽 위 모서리에 표시됨을 나타냅니다. 
spread.options.backgroundImageLayout = GC.Spread.Sheets.ImageLayout.stretch;

3. grayAreaBackColor를 사용하여 시트 뒤에 있는 회색 영역의 배경색을 설정할 수도 있습니다. 

spread.options.grayAreaBackColor = 'gray';
