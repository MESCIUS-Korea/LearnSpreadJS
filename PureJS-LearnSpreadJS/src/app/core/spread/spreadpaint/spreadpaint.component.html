Spread 페인트 | Spread Paint

1. Spread를 변경 한 후에는 구성 요소가 이러한 변경 사항을 처리하고 업데이트해야 합니다.


2. 다시 그리기를 사용하여 Spread를 다시 칠할 수도 있습니다.

var spread  = $('#ss').data('workbook');
spread.repaint();

3. 대부분의 경우 Spread를 변경하면 구성 요소가 자동으로 새로 고칩니다. 한 번에 많은 변경을 실시해, Spread가 여러 차례 다시 칠해지기를 원하지 않는 경우, suspendPaint를 호출 해 변경을 실시하면서 페인트 다시 그만 둘 수 있습니다. 변경이 끝나면 resumePaint를 호출하여 재 페인트된 Spread를 알 수 있습니다.
예를 들어: 

var spread  = $('#ss').data('workbook');
spread.suspendPaint(); 
spread.addSheet(0);
spread.fromJSON(json); // json data
spread.resumePaint();

샘플코드:

결과화면과 함께 HTML, JS 코드도 확인이 가능합니다.
