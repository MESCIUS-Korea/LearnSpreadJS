문서테마 | Document Theme

1. Spread.Sheets은 글꼴 및 색상 설정을 포함하여 전반적인 스타일을 정의 할 수 있는 테마를 지원합니다.

2. Spread.Sheets은 몇 가지 기본 테마를 제공합니다. currentTheme 메서드를 사용하여 시트의 현재 테마를 가져오고 설정합니다 예 : 

var spread = $('#ss').data('workbook');
var sheet = spread.getActiveSheet();
sheet.currentTheme(GC.Spread.Sheets.SpreadThemes.civic);

3. 이 예제에서와 같이 테마 색상이나 표준 색상을 사용하여 셀의 foreColor 또는 backColor 메서드를 설정할 수 있습니다.

var spread = $('#ss').data('workbook');
var sheet = spread.getActiveSheet();
sheet.getCell(1,1).foreColor('Accent 1 50');
sheet.getCell(1,1).backColor('Background 2 80');
